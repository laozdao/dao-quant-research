# 煤炭行业供需格局与价格预测

## 三、煤炭价格预测模型

### 3.1 模型构建

#### 多元回归模型

$$P_t = \beta_0 + \beta_1 S_t + \beta_2 D_t + \beta_3 I_t + \beta_4 M_t + \epsilon_t$$

其中：
- $P_t$：煤炭价格
- $S_t$：供给指标
- $D_t$：需求指标
- $I_t$：库存指标
- $M_t$：宏观指标

#### 变量选择

| 变量 | 指标 | 预期符号 |
|------|------|---------|
| 供给 | 产量增速 | - |
| 需求 | 火电发电量增速 | + |
| 库存 | 港口库存 | - |
| 宏观 | GDP增速 | + |
| 替代 | 水电增速 | - |

## 六、总结与展望

### 6.1 核心结论

1. **供需紧平衡**：产能约束下供需维持紧平衡
2. **价格中枢上移**：成本支撑价格中枢上移
3. **高股息属性**：现金流充裕，分红能力强
4. **估值偏低**：当前估值处于历史低位

### 6.2 投资建议

**长期投资者**：
- 配置高股息龙头
- 享受分红收益
- 忽略短期波动

**主动投资者**：
- 把握煤价周期
- 关注政策变化
- 动态调整仓位

### 6.3 未来展望

1. **产能约束**：新增产能有限
2. **需求韧性**：能源安全需要
3. **转型压力**：长期面临双碳约束
4. **分红持续**：现金流支撑高分红
