# 工业气体全景与市场格局：万亿美元基础材料的量化分析

**系列定位：工业气体深度研究 第1篇（总览篇）**

> 工业气体被誉为"现代工业的血液"，电子特气被称为"半导体的粮食"。在一个年规模超过2000亿美元、中国占比持续攀升的庞大市场中，气体虽无形，却支撑着从钢铁冶炼到芯片制造的几乎所有工业门类。本篇报告以量化视角，全景扫描这一万亿级基础材料市场的格局、结构与机遇。

## 第1章 工业气体行业全景定义与分类体系

### 1.1 行业定义与战略定位

工业气体（Industrial Gases）是指在工业生产过程中，以气体形态存在、具有特定物理化学性质、广泛应用于各类工业制程的基础性材料。从最基础的大气分离气体到纯度高达99.9999%（6N）以上的电子特种气体，工业气体构成了现代工业体系不可或缺的底层材料。

**战略定位的三个维度：**

| 维度 | 定位 | 核心逻辑 |
|------|------|----------|
| 经济维度 | "现代工业的血液" | 覆盖冶金、化工、电子、医疗等几乎所有工业门类 |
| 技术维度 | "半导体的粮食" | 电子特气占晶圆制造材料成本的13%，前三大材料之一 |
| 安全维度 | "战略储备资源" | 氦气等稀有气体关乎国防与高端制造，供应受地缘政治深刻影响 |

工业气体行业的核心特征可归纳为"三高一长"：

- **高壁垒**：核心提纯技术被少数国际巨头垄断，电子特气客户认证周期长达1-2年
- **高粘性**：现场制气（On-site）模式合同期10-15年，客户切换成本极高
- **高集中**：全球前四大巨头占据超过50%的市场份额
- **长周期**：空分装置从投建到投产周期长，产能扩张具有刚性

### 1.2 完整分类体系

工业气体可按纯度、用途、来源等多个维度进行分类。本报告采用"四级分类法"，建立完整的行业分类框架。

**第一级：按纯度与技术层级分类**

```
工业气体
├── 大宗气体（Bulk Gases）—— 纯度99%-99.999%，基础工业用途
├── 电子大宗气体（Electronic Bulk Gases）—— 纯度5N-6N，半导体/面板
├── 电子特种气体（Electronic Specialty Gases）—— 纯度4N-7N，特定工艺用途
└── 稀有气体（Rare Gases）—— 大气中极低丰度，战略稀缺性
```

| 类别 | 核心品种 | 化学式 | 主要应用 |
|------|----------|--------|----------|

**第三级：按应用场景分类**

```
└── 特殊应用（航天、国防、科研）
```

### 1.3 技术壁垒金字塔

工业气体的技术壁垒呈现明显的金字塔结构，从底层大宗气体到顶层电子特气/稀有气体，纯度要求每提升一个数量级，技术难度呈指数级增长。

```
                    ┌──────┐
                    │ 7N+  │ 稀有气体超纯化
                    ├──────┤
                    │ 6N   │ 电子特种气体（NF₃、WF₆等）
               技术 ├──────┤
              壁垒  │ 5N   │ 电子大宗气体
              指数  ├──────┤
              增长  │ 4N   │ 高纯工业气体
                    ├──────┤
                    │ 3N   │ 标准工业气体
                    ├──────┤
                    │ 2N   │ 普通工业气体
                    └──────┘
```

**纯度等级说明：**
- N（Nine）代表数字"9"的个数，如5N即99.999%
- 集成电路制造通常要求4N-6N纯度，先进制程要求更高
- 杂质控制达到ppb（十亿分之一）甚至ppt（万亿分之一）级别

### 1.4 行业发展的三大驱动力

**驱动力一：半导体产业周期与技术迭代**

全球半导体产业正经历前所未有的扩张周期。AI大模型训练与推理对算力的指数级需求，直接驱动了晶圆厂产能的大规模扩张。每新建一座12英寸晶圆厂，其对电子气体的年需求量约数千万立方米，设备投资中气体供应系统占比约5%-8%。

**驱动力二：新能源革命与氢经济崛起**

绿氢作为碳中和路径中的关键环节，正在催生万亿级市场机遇。工业氢气从灰氢（化石能源制取）向蓝氢（CCUS）和绿氢（电解水）的转型，将重塑整个氢气产业链的成本结构和竞争格局。

**驱动力三：国产替代与供应链安全**

地缘政治博弈加剧背景下，电子特气作为半导体制造的核心耗材，其国产化率已成为衡量产业链安全的关键指标。2025年日本对六氟化钨（WF₃）和三氟化氮（NF₃）的断供事件，加速了中国电子特气国产替代进程。

## 第3章 产业链全景图谱

### 3.1 产业链总览

工业气体产业链可分为六大环节：原料获取 → 气体分离/合成 → 提纯精制 → 充装储存 → 运输配送 → 终端应用。每个环节的技术壁垒和附加值差异显著。

```
┌─────────┐   ┌──────────┐   ┌──────────┐   ┌─────────┐   ┌─────────┐   ┌──────────┐
│  原料    │ → │  分离/   │ → │  提纯    │ → │  充装   │ → │  运输   │ → │  终端    │
│  获取    │   │  合成    │   │  精制    │   │  储存   │   │  配送   │   │  应用    │
└─────────┘   └──────────┘   └──────────┘   └─────────┘   └─────────┘   └──────────┘
  空气         空分(ASU)     吸附法         钢瓶/集装格   管道输送      半导体制造
  天然气       深冷分离      膜分离         杜瓦罐        液槽车        显示面板
  化工副产     水电解        催化转化        ISO罐         长管拖车      冶金化工
  工业副产     化学合成      低温精馏       气柜储气      船舶运输      医疗健康
```

### 3.2 上游：原料与分离技术

**空气分离（Air Separation Unit, ASU）**：工业气体的最主要来源。通过深冷精馏技术将空气液化后分离出O₂、N₂、Ar等组分。

| 空分装置等级 | 制氧能力 | 技术拥有方 | 应用场景 |
|-------------|---------|-----------|----------|
| 大型（>6万m³/h） | 60000+ | Linde, Air Liquide, 杭氧 | 钢铁、化工一体化 |
| 中型（1-6万m³/h） | 10000-60000 | 国内外均有 | 大中型制造 |
| 小型（<1万m³/h） | <10000 | 国内为主 | 中小企业用气 |

> **国产化突破：** 2010年之前，国内大型空分装置基本被国外巨头垄断。杭氧股份作为国内空分龙头，从首套3万级开始攻克，到6万级、8万级，再到10万级，每一次突破都大幅降低了国内企业对进口设备的依赖。

**天然气提氦**：全球约70%的氦气从含氦天然气中提取。核心技术为膜分离+低温精馏的组合工艺。

**化工副产提取**：焦炉煤气提氢、氯碱副产氢、合成氨尾气提氩等，是低成本获取大宗气体的有效途径。

### 3.3 中游：提纯与制备

提纯环节是工业气体产业链的核心技术壁垒，也是决定气体纯度和品质的关键环节。

| 提纯技术 | 原理 | 适用气体 | 纯度上限 |
|----------|------|----------|---------|
| 低温精馏 | 利用沸点差分离 | O₂/N₂/Ar/稀有气体 | 6N+ |
| 变压吸附（PSA） | 分子筛选择性吸附 | N₂/O₂/H₂ | 5N |
| 膜分离 | 分子大小选择性透过 | H₂/O₂/N₂ | 4N |
| 催化反应 | 化学转化去除杂质 | 电子特气 | 6N+ |
| 热分解 | 高温分解杂质 | SiH₄/NF₃等 | 6N+ |

### 3.4 中游：充装与储存

**主流充装方式：**

| 方式 | 容量范围 | 压力 | 适用气体 | 运输成本 |
|------|---------|------|---------|---------|
| 钢瓶 | 40L/瓶 | 15MPa | 所有气体 | 高 |
| 集装格 | 16瓶/组 | 15MPa | 大宗气体 | 中高 |
| 杜瓦罐 | 100-1000L | 低压/常压 | 液态气体 | 中 |
| ISO罐 | 10-25吨 | 常压 | 液态大宗 | 中低 |
| 管道输送 | 连续 | 管网压力 | 现场制气 | 最低 |

### 3.5 下游：核心应用领域

**半导体制造（最高附加值领域）：**

集成电路制造过程中使用超过**110种**电子气体，贯穿光刻、刻蚀、薄膜沉积（CVD/PECVD）、离子注入、外延生长、清洗等几乎所有关键工序。

| 工序 | 核心气体 | 技术要求 |
|------|---------|---------|
| 刻蚀 | CF₄/SF₆/Cl₂/HBr/NF₃ | 选择比、均匀性 |
| CVD薄膜 | SiH₄/TEOS/WF₆/NH₃/N₂O | 致密性、台阶覆盖 |
| 离子注入 | BF₃/AsH₃/PH₃ | 注入深度控制 |
| 外延生长 | SiH₄/HCl/H₂ | 晶格匹配 |
| 清洗 | NF₃/SF₆/超临界CO₂ | 无损伤清洗 |

> **超临界CO₂清洗：** 高纯CO₂是HBM、3D NAND等先进制程超临界清洗的唯一成熟方案。在先进制程中，晶圆线路间距已进入纳米级，传统湿法清洗难以深入微细结构，超临界CO₂因其极低的表面张力（接近零）和优异的渗透能力，成为解决高深宽比结构清洗的关键技术路径。

**显示面板制造：**

TFT-LCD和OLED面板制造过程大量使用电子气体，主要包括：

| 工序 | 核心气体 |
|------|---------|
| TFT成膜（PECVD） | SiH₄/NH₃/N₂O/PH₃ |
| 刻蚀 | NF₃/SF₆/CF₄ |
| 掩膜版清洗 | NF₃ |
| OLED蒸镀 | 高纯N₂/Ar（保护气） |

### 3.6 商业模式分析

工业气体行业的商业模式呈现显著的分层特征：

## 第5章 电子特种气体——工业气体"塔尖"产品

### 5.1 电子特气行业定义与特征

电子特种气体（Electronic Specialty Gases）是指在半导体、显示面板等电子制造领域使用的、具有特定化学特性、高纯度（通常4N-7N）、对杂质控制要求极高的特种气体。

**电子特气的核心特征：**

1. **纯度极高**：通常要求99.99%（4N）至99.99999%（7N），部分先进制程要求更高
2. **品种繁多**：集成电路制造使用超过110种电子气体
3. **用量少但价值高**：单个品种年用量可能仅数十吨，但单价可达数万元/公斤
4. **认证壁垒极高**：从送样到量产通常需要1-2年，期间经过多轮验证
5. **供应商切换成本高**：一旦通过认证，客户极少更换供应商

### 5.2 电子特气市场量化分析

**全球电子特气市场规模：**

**中国电子特气市场：**

### 5.3 技术壁垒分析

电子特气的技术壁垒主要体现在以下四个维度：

**维度一：合成/制备技术**

电子特气的合成路线复杂，反应条件苛刻（高温、高压、腐蚀性介质），且合成过程中需要严格控制副产物和杂质。

**维度二：提纯精制技术**

将粗品气体提纯至电子级（5N-7N）需要多级提纯工艺的组合，包括低温精馏、吸附、膜分离、催化转化等，不同气体的提纯工艺差异极大。

**维度三：分析检测技术**

电子气体的纯度分析需要检测ppb甚至ppt级别的杂质，需要使用GC-MS、ICP-MS、FTIR等高端分析仪器，建立完整的分析方法体系。

**维度四：包装与运输技术**

电子特气对包装容器的材质、内表面处理、阀门密封等要求极高，部分腐蚀性/毒性气体需要专用的包装和运输方案。

### 5.4 客户认证体系

电子特气进入晶圆厂供应链的认证流程是行业最高壁垒之一：

```
实验室评估 → 小试送样 → 中试验证 → 小批量试用 → 量产导入
   (3-6个月)   (3-6个月)   (3-6个月)    (6-12个月)   (持续监控)
```

**认证周期合计：1-2年**

各阶段的关键要求：
1. **实验室评估**：杂质全谱分析、包装兼容性测试
2. **小试送样**：在非关键工序上小批量试用
3. **中试验证**：在真实生产线上进行制程验证
4. **小批量试用**：逐步扩大用量，验证批次稳定性
5. **量产导入**：正式成为合格供应商，持续审计

> 这意味着电子特气企业一旦通过认证，将享有长期稳定的订单，新进入者极难在短期内撼动既有格局。但反过来，如果产品品质出现波动被取消资格，重新认证又将面临漫长的等待。

### 5.5 超临界CO₂清洗：技术前沿与新机遇

高纯CO₂在半导体先进制程中的超临界清洗应用，是近年来电子气体领域最值得关注的技术创新之一。

**超临界CO₂清洗技术原理：**
- CO₂在31.1°C、7.38MPa以上进入超临界状态
- 超临界流体兼具气体的高渗透性和液体的溶解力
- 表面张力接近零，可深入高深宽比微结构进行清洗

**核心应用场景：**

| 应用 | 技术要求 | 市场前景 |
|------|---------|---------|
| HBM（高带宽内存） | 高深宽比TSV结构清洗 | 需求爆发式增长 |
| 3D NAND | 多层堆叠通道清洗 | 层数增加驱动需求 |
| 先进制程（<7nm） | 纳米级特征尺寸清洗 | 不可替代性增强 |

> **关键判断：** 超临界CO₂清洗是HBM和3D NAND先进制程目前唯一成熟的超深宽比结构清洗方案。随着AI算力需求驱动HBM产能大规模扩张，高纯CO₂的需求量有望在2025-2027年迎来爆发式增长。

### 6.1 稀有气体的稀缺性与战略价值

稀有气体（氦、氖、氪、氙）因其在大气中极低的丰度和独特的物理化学性质，成为工业气体领域中稀缺性最高、战略价值最大的子品类。

| 氪气（Kr） | ★★★ | ★★★ | 中 | ~5000-10000元/立方米 |

### 6.2 氦气：全球最紧缺的战略气体

氦气是稀有气体中应用最广、供应最紧张的品种。全球氦资源高度集中，供应链脆弱性突出。

**全球氦气供应格局：**

| 俄罗斯 | ~15% | 阿穆尔气田等 | 受制裁影响 |
| 其他 | ~20% | 澳大利亚、波兰等 | 不确定 |

> **中国氦气安全评估：** 中国氦资源对外依存度超过95%，是全球最大的氦气进口国之一。国内氦气主要从美国、卡塔尔进口，一旦地缘政治紧张，氦气供应可能面临断供风险。近年来，中国正在加快伴生气提氦技术的突破，以期降低对外依存度。

### 6.3 氪气与氙气：高价值稀缺资源

氪气和氙气在大气中的丰度极低（氪1.14ppm，氙0.087ppm），是空气分离中提取难度最大的气体品种。

**氙气（Xe）—— "液体黄金"**

氙气因其独特的发光特性和麻醉特性，单价极高：
- 半导体光刻：深紫外（DUV）光源需要氙气
- 照明：高端车灯、影院放映
- 医疗：麻醉剂（Xe麻醉具有神经保护作用）
- 航天：离子推进器工质

**氖气（Ne）—— 光刻与激光的核心耗材**

氖气是DUV光刻机（ArF准分子激光器）的关键工作气体，每台DUV光刻机每年消耗数万升氖气。随着全球DUV光刻机保有量持续增加，氖气需求稳步增长。

> **供应链风险警示：** 2022年俄乌冲突导致全球氖气供应紧张（乌克兰Ingas和Cryoin公司曾占全球氖气产量约50%），价格短期内暴涨数倍。这一事件凸显了稀有气体供应链的地缘政治脆弱性。

### 6.4 稀有气体市场趋势

**供需趋势判断：**

1. **需求端加速增长**：半导体先进制程、HBM/3D NAND大规模扩产、航天产业扩张共同驱动稀有气体需求
2. **供给端刚性约束**：稀有气体主要从空分副产或天然气中提取，产能扩张受制于上游装置规模，弹性有限
3. **价格中枢上移**：在供需偏紧格局下，稀有气体价格中枢有望持续上移
4. **国产替代迫在眉睫**：中国对氦气的极度依赖构成供应链安全隐患，国产提氦技术突破具有战略意义
