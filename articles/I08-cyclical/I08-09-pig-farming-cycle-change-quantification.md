## 一、研究背景与核心问题

### 1.1 传统猪周期的运行逻辑

生猪养殖行业作为典型的周期性行业，其价格波动规律在过去二十年间形成了较为稳定的运行范式。传统"猪周期"通常指以3-4年为完整一轮的价格波动周期，底层逻辑根植于生猪养殖的生物学特性：能繁母猪配种妊娠需114天，仔猪育肥至出栏需约180天，从母猪补栏到商品猪出栏全周期约10-12个月。这种生产时滞导致养殖户决策基于当期价格而非未来价格，形成典型的"蛛网模型"动态：猪价上涨→盈利→补栏增加→10-12个月后供给过剩→猪价下跌→亏损→淘汰母猪→10-12个月后供给收缩→猪价回升。

在传统猪周期框架下，每一轮周期都呈现清晰的四阶段特征：猪价触底回升→养殖盈利扩张→产能快速补充→供给过剩下跌。价格振幅通常极为剧烈，高峰与低谷价差可达100%-200%，2007年、2011年、2016年、2020年猪价峰值分别达14.9元/公斤、19.9元/公斤、21.0元/公斤、38.0元/公斤，周期低谷往往跌破成本线，迫使行业通过深度亏损实现产能去化。

### 1.2 2026年猪周期"变天"的核心命题

进入2026年，生猪养殖行业呈现出与传统周期截然不同的运行特征。6月份外三元生猪均价跌至9.5-9.8元/公斤，创近十年新低，但行业并未出现以往周期低谷期那种恐慌性的母猪淘汰和大规模清栏。能繁母猪存栏量在2026年第一季度仍维持3904万头的高位，仅较峰值温和去化3-5%，去化速度明显慢于历史同期。更为反常的是，头部养殖企业在深度亏损环境下不仅未收缩产能，反而凭借充裕资金储备维持甚至扩大出栏规模——2025年牧原股份出栏7798万头、温氏股份出栏4048万头，前20家上市猪企合计出栏占比达36%。

这一现象引发市场关于"猪周期是否消失"的激烈争论。传统周期判断指标——能繁母猪存栏、猪粮比、仔猪价格——的预测效力显著下降：能繁母猪存栏已连续9个月环比下降，但猪价并未如期企稳回升；猪粮比长期在4.0:1预警线下方运行，行业自繁自养头均亏损超300元，但产能去化幅度远不及2014年、2018年周期底部的10%-15%。

核心命题由此浮现：**为什么以能繁母猪存栏为核心的老规律失效了？猪周期的底层传导机制发生了哪些结构性变化？在新的行业格局下，如何判断周期拐点、评估投资价值？** 本报告试图通过量化分析框架，系统回答上述问题。

### 1.3 研究框架：PSY效率因子×规模化因子×政策调控因子

为解析2026年猪周期的"变天"现象，本报告构建了三因子分析框架：

**PSY效率因子（繁殖效率）**：每头能繁母猪年提供断奶仔猪数（PSY）从2010年代的18头提升至当前的24-25头，头部企业更达30头以上。同等能繁母猪存栏水平下，实际产能供给能力提升了33%-67%，形成巨大的"隐形产能"，使得能繁母猪存栏指标对实际供给的指示意义大幅下降。

**规模化因子（产业集中度）**：2025年规模猪场出栏占比已超70%，前20家上市猪企市占率达36%。规模化企业凭借资本市场融资能力、精细化成本管理和抗风险能力，在亏损周期中表现出显著的"抗去产能"特征，改变了传统周期中"亏损即去产能"的运行逻辑。

**政策调控因子（政府干预）**：能繁母猪正常保有量从3900万头下调至3750万头，储备肉收储/投放机制常态化，年出栏10万头以上集团纳入重点监测。政策导向从"应急调控"转向"预期管理"，形成"托底不抬顶"的调控新范式，有效平抑了价格的极端波动。

三因子的交互作用，共同塑造了2026年猪周期"深度磨底、窄幅波动、弱周期运行"的新特征。本报告将围绕这一框架，逐层展开量化分析。

## 二、猪周期历史规律与传导机制

### 2.1 完整猪周期的四阶段定义

基于2006年至2026年四轮完整周期的历史数据，传统猪周期的四个阶段具有明确的量化界定标准：

| 阶段 | 名称 | 猪价特征 | 养殖盈利 | 产能行为 | 持续时间 |
|------|------|----------|----------|----------|----------|
| 第一阶段 | 触底回升期 | 猪价低于成本线，跌至周期底部 | 深度亏损，头均亏损200-500元 | 能繁母猪加速淘汰，产能去化5%-15% | 6-12个月 |
| 第二阶段 | 盈利扩张期 | 猪价快速上涨，突破成本线 | 盈利扩大，头均盈利300-1000元 | 补栏启动，能繁母猪触底回升 | 8-14个月 |
| 第三阶段 | 产能扩张期 | 猪价高位震荡或缓慢回落 | 盈利丰厚，头均盈利500-2000元 | 能繁母猪快速增加，产能扩张10%-20% | 10-16个月 |
| 第四阶段 | 供给过剩期 | 猪价快速下跌，跌破成本线 | 盈利收窄至亏损，头均亏损100-400元 | 能繁母猪高位运行后去化 | 8-12个月 |

四阶段合计构成一轮完整周期，历时通常为36-48个月。其中，从能繁母猪存栏见顶到猪价触底约为10-12个月，从能繁母猪见底到猪价见顶同样约为10-12个月，这是由生猪养殖的生物学周期所决定的刚性时滞。

### 2.2 核心传导链条：能繁母猪存栏→10个月后生猪出栏→猪价波动

传统猪周期的核心传导机制为：**能繁母猪存栏变化率（t月）→仔猪出生量变化率（t+0至t+4月）→育肥猪出栏量变化率（t+10至t+12月）→市场供给变化率（t+10月）→猪价变化（t+10至t+14月）**。

能繁母猪存栏量的变化需经历约10-12个月时滞才能传导至猪价：配种妊娠0-4个月，仔猪断奶约1个月，育肥出栏5-6个月。在这一传导框架下，能繁母猪存栏是判断未来10-12个月猪价走势的领先指标，历史相关系数约为-0.65至-0.75。

然而，这一相关性的成立隐含了关键假设：**单位能繁母猪的产出效率（PSY）保持相对稳定**。当PSY发生结构性跃升时，能繁母猪存栏与生猪出栏之间的数量关系被打破，传统传导链条的预测效力大幅下降。

### 2.3 历史四轮猪周期复盘（2006-2026）

#### 第一轮周期（2006年6月—2010年4月，约46个月）

| 指标 | 数据 |
|------|------|
| 周期起点 | 2006年6月，猪价5.8元/公斤 |
| 周期峰值 | 2008年3月，猪价17.4元/公斤 |
| 周期终点 | 2010年4月，猪价9.0元/公斤 |
| 价格振幅 | 峰值/谷底=3.0倍 |
| 驱动因素 | 2006年高致病性蓝耳病暴发，产能骤降20%；2008年奥运会需求拉动；2009年甲型H1N1流感（猪流感）误伤消费 |
| 产能变化 | 能繁母猪从4300万头降至3800万头，去化12.8% |
| 周期特征 | 疫病主导，价格波动剧烈，消费事件扰动 |

第一轮周期是典型"疫病驱动型"周期。2006年高致病性猪蓝耳病暴发导致产能骤减约20%，猪价从5.8元/公斤飙升至17.4元/公斤。随后暴利刺激产能恢复，加上2009年甲型H1N1流感被误称"猪流感"导致消费恐慌，猪价快速回落至9.0元/公斤。

#### 第二轮周期（2010年4月—2014年4月，约48个月）

| 指标 | 数据 |
|------|------|
| 周期起点 | 2010年4月，猪价9.0元/公斤 |
| 周期峰值 | 2011年9月，猪价19.9元/公斤 |
| 周期终点 | 2014年4月，猪价10.5元/公斤 |
| 价格振幅 | 峰值/谷底=2.2倍 |
| 驱动因素 | 2010年冬季低温导致仔猪腹泻高发；2011年通胀周期叠加需求增长；2012-2013年产能持续扩张，供给过剩 |
| 产能变化 | 能繁母猪从4700万头增至5078万头（2012年峰值），后降至4390万头，去化13.6% |
| 周期特征 | 成本推动+需求拉动，规模化开始萌芽 |

第二轮周期中，猪价创下19.9元/公斤的历史新高。推动因素包括2010年冬季低温导致仔猪腹泻、通胀环境下的成本推动，以及肉类消费持续增长。值得注意的是，这一轮周期中规模化养殖开始加速，但散户仍占主导地位（规模化率约40%）。周期底部产能去化幅度达13.6%，行业经历深度亏损。

#### 第三轮周期（2014年4月—2018年5月，约49个月）

| 指标 | 数据 |
|------|------|
| 周期起点 | 2014年4月，猪价10.5元/公斤 |
| 周期峰值 | 2016年6月，猪价21.0元/公斤 |
| 周期终点 | 2018年5月，猪价10.0元/公斤 |
| 价格振幅 | 峰值/谷底=2.0倍 |
| 驱动因素 | 2014年《畜禽规模养殖污染防治条例》实施，环保拆迁淘汰散户产能；2015年仔猪流行性腹泻；2016年禁养区清退高峰 |
| 产能变化 | 能繁母猪从4390万头降至3680万头（2016年低点），去化16.2% |
| 周期特征 | 环保政策主导，规模化率提升至55%，周期振幅开始收窄 |

第三轮周期是政策干预深度影响周期的开端。2014年环保条例实施、2015年"水十条"落地，环保禁养区清退导致大量散户退出。环保因素导致被动去产能约8%-10%，叠加疫病因素，能繁母猪去化幅度达16.2%，创历史最大降幅。猪价在2016年6月达到21.0元/公斤高点。周期底部（2018年5月）猪价10.0元/公斤，规模化率提升至55%。

#### 第四轮周期（2018年5月—2022年3月，约46个月）

| 指标 | 数据 |
|------|------|
| 周期起点 | 2018年5月，猪价10.0元/公斤 |
| 周期峰值 | 2019年11月-2020年2月，猪价38.0-40.0元/公斤 |
| 周期终点 | 2022年3月，猪价12.0元/公斤 |
| 价格振幅 | 峰值/谷底=3.8倍 |
| 驱动因素 | 2018年8月非洲猪瘟（ASF）传入中国，产能断崖式下跌；2019-2020年超级暴利周期；2021年产能快速恢复，猪价暴跌 |
| 产能变化 | 能繁母猪从3400万头（2018年低点）降至2019年9月的1913万头，降幅43.8%；后快速恢复至2021年6月的4564万头 |
| 周期特征 | 非瘟主导的超级周期，历史最大振幅，规模化率跃升至65% |

第四轮周期是历史上最极端的周期。2018年8月非洲猪瘟传入后，能繁母猪存栏从3350万头暴跌至2019年9月的1913万头，降幅43.8%。供给崩塌推动猪价在2020年2月达到38.0-40.0元/公斤历史极值，自繁自养头均盈利超2500元。超级暴利刺激史无前例的产能扩张：能繁母猪存栏20个月内增长138%。然而，产能过度恢复导致2021年猪价从36元/公斤暴跌至10元/公斤，2022年3月跌至12.0元/公斤，行业陷入深度亏损。

#### 第五轮周期（2022年3月—至今，已运行约40个月）

| 指标 | 数据 |
|------|------|
| 周期起点 | 2022年3月，猪价12.0元/公斤 |
| 周期峰值 | 2022年10月-2023年1月，猪价26.0-28.0元/公斤 |
| 当前阶段 | 2026年6月，猪价9.5-9.8元/公斤 |
| 当前运行状态 | 深度磨底阶段，已持续约16个月 |
| 驱动因素 | 2022年产能去化不充分；2023-2024年PSY提升导致供给持续宽松；规模化企业抗亏损能力强，去化缓慢 |
| 产能变化 | 能繁母猪从2022年12月的4390万头增至2024年中的4042万头，后降至2026年Q1的3904万头，累计去化3.5% |
| 周期特征 | 弱周期、窄幅波动，传统规律失效 |

第五轮周期呈现出与历史周期截然不同的运行特征。2022年3月的周期底部，能繁母猪存栏仅去化约8%（从4564万头降至4390万头），远不及历史周期底部10%-15%的去化幅度。随后猪价反弹至26-28元/公斤，但仅维持约4个月即快速回落。2023年至2026年，猪价长期在成本线下方运行，但产能去化极为缓慢——能繁母猪存栏从2024年中的高点4042万头降至2026年Q1的3904万头，16个月累计仅去化3.5%。

### 2.4 传统周期判断指标及其在2026年的失效

#### 指标一：能繁母猪存栏

能繁母猪存栏指标的失效源于PSY的结构性跃升。2014年行业平均PSY约为18头，2024年提升至24-25头，头部企业达30头。这意味着2026年Q1的3904万头能繁母猪，其理论产能供给相当于2014年的5200-5400万头（按PSY标准化折算）。能繁母猪存栏的"名义值"与"实际产能"之间的缺口持续扩大，导致该指标的领先预测效力大幅下降。

#### 指标二：猪粮比

猪粮比（生猪出栏价/玉米批发价）是衡量养殖盈利水平和政策干预阈值的核心指标。根据《缓解生猪市场价格周期性波动调控预案》：

2026年6月猪粮比约4.0:1，已处于重度亏损区间。按照传统规律，如此深度的亏损将触发大规模的产能去化。但当前周期中，猪粮比低于5.0:1已持续超过12个月，能繁母猪去化幅度仅为3.5%，显著低于2014年周期底部（猪粮比4.5:1持续8个月，去化13.6%）和2018年周期底部（猪粮比4.8:1持续6个月，去化10.2%）。

#### 指标三：仔猪价格

仔猪价格是反映养殖户补栏意愿的领先指标。2026年6月，15公斤仔猪价格约350-400元/头，处于盈亏平衡附近。散户补栏意愿低迷与规模化企业维持补栏形成分化——温氏、新希望等企业仍保持正常的种猪更新和仔猪投栏节奏，导致仔猪价格未出现历史底部那种"无人问津"的极端低值，传统指标的信号意义被削弱。

## 三、2026年猪周期现状：深度磨底的全景数据

### 3.1 猪价：创近十年新低的极端磨底

2026年6月全国外三元生猪出栏均价9.5-9.8元/公斤，创2016年以来近十年新低。价格较2022年10月周期高点28.0元/公斤下跌65.0%，较行业平均完全成本13.0元/公斤低24.6%-26.9%。

| 时间 | 外三元猪价（元/公斤） | 环比变化 | 同比变化 |
|------|----------------------|----------|----------|
| 2025年1月 | 14.5 | -5.2% | -8.0% |
| 2025年6月 | 11.8 | -18.6% | -22.0% |
| 2025年12月 | 10.2 | -13.6% | -28.0% |
| 2026年3月 | 9.8 | -3.9% | -15.0% |

区域分布上，北方产区猪价普遍低于南方销区0.5-1.0元/公斤。猪价持续低迷的核心原因在于供给宽松：2026年上半年全国生猪出栏量约3.55亿头，同比增长约2.8%，而消费需求同比仅增长0.5%-1.0%，供过于求幅度约1.5-2.0个百分点。

### 3.3 猪粮比：长期跌破预警线的深度亏损

2026年6月猪粮比约4.0:1，已持续低于5.5:1收储预警线约14个月，低于4.5:1重度亏损线约8个月。按照行业平均完全成本13.0元/公斤计算，2026年6月自繁自养头均亏损约为402元/头（以9.65元/公斤猪价、120公斤出栏体重计算）。规模化企业完全成本11.5-12.0元/公斤，头均亏损160-280元/头；散户成本14.0元/公斤以上，头均亏损超520元/头。

## 四、猪周期"变天"的三大结构性变化

2026年猪周期呈现出与传统周期截然不同的运行特征，其背后是三大结构性变化的深层作用：繁殖效率跃升（PSY因子）、规模化主导（集中度因子）和政策调控常态化（政策因子）。三者的交互作用，从根本上重塑了猪周期的传导机制、波动幅度和运行节奏。

### 4.1 变化一：规模化主导——产业格局的质变

#### 4.1.1 规模化率的快速提升

中国生猪养殖行业的规模化进程在2018年非洲猪瘟后显著加速。散户因防疫能力弱、资金实力差，在疫情冲击下大量退出市场；而规模化企业凭借资本优势和技术优势，趁势扩张产能。

| 年份 | 年出栏500头以上规模场占比 | 年出栏5000头以上规模场占比 | 前20家上市猪企市占率 |
|------|--------------------------|--------------------------|---------------------|
| 2010 | 34% | 12% | 3% |
| 2014 | 42% | 18% | 5% |
| 2018 | 55% | 28% | 8% |
| 2020 | 65% | 38% | 15% |
| 2022 | 68% | 45% | 25% |
| 2024 | 70% | 52% | 33% |
| 2025 | >70% | 58% | 36% |

2025年，年出栏500头以上规模场出栏占比已超过70%，其中年出栏5000头以上的中大型规模场占比达到58%。前20家上市猪企合计出栏约2.56亿头，占全国总出栏量的36%，较2018年的8%提升了28个百分点。

#### 4.1.2 规模化企业的"抗去产能"特征

规模化企业在深度亏损周期中的行为模式，与散户存在本质差异：

| 对比维度 | 散户/中小规模场 | 规模化企业 |
|----------|----------------|------------|
| 资金实力 | 自有资金投入为主，融资渠道有限 | 资本市场融资（定增、可转债、公司债、银行贷款） |
| 亏损承受能力 | 现金流断裂即退出，通常亏损3-6个月即清栏 | 可承受12-24个月连续亏损，凭融资维持运营 |
| 母猪处置决策 | 猪价跌破成本即淘汰母猪，减少存栏 | 将母猪视为核心资产，优先保种群、降成本 |
| 生产性生物资产 | 变现能力强，退出门槛低 | 固定资产投入大，退出沉没成本高 |
| 信息获取 | 基于当期价格和周边养殖户行为决策 | 基于行业数据、期货价格、政策导向进行前瞻性决策 |
| 疫病防控 | 防疫设施简陋，疫病损失大 | 生物安全体系完善，疫病损失可控 |

规模化企业的"抗去产能"行为直接表现为在深度亏损环境下仍维持甚至扩大产能。以牧原股份为例，2024-2025年猪价长期处于成本线下方，但公司出栏量仍从7180万头增长至7798万头，同比增长8.5%。温氏股份出栏量从3848万头增长至4048万头，同比增长5.2%。这种"越亏越扩"的行为在传统周期中极为罕见，但在资本驱动型扩张逻辑下成为常态。

#### 4.1.3 散户加速出清与规模化填补

在2024-2026年深度亏损周期中，散户和中小规模场加速退出市场。据行业估算，2024年至2026年Q1散户数量减少约15%-20%，对应产能去化约8%-10%。然而，头部企业通过产能扩张和效率提升，部分抵消了散户退出的供给收缩效应。

| 群体 | 2024年出栏占比 | 2026年Q1出栏占比 | 变化 | 产能行为 |
|------|---------------|-----------------|------|----------|
| 前10家上市猪企 | 28% | 33% | +5pct | 扩张 |
| 第11-50家中型规模场 | 22% | 24% | +2pct | 微增或持平 |
| 第51-200家小型规模场 | 18% | 16% | -2pct | 收缩 |
| 散户（年出栏<500头） | 32% | 27% | -5pct | 加速退出 |

散户退出的产能缺口约60%被头部企业扩张填补，40%转化为实际供给收缩。即便散户去化幅度达到历史水平，行业总供给收缩幅度也仅相当于传统周期的40%-50%，猪价回升力度和速度相应减弱。

### 4.2 变化二：繁育效率跃升——"隐形产能"的释放

#### 4.2.1 PSY的结构性提升

每头能繁母猪年提供断奶仔猪数（PSY）是衡量繁殖效率的核心指标。过去十年间，中国生猪养殖行业的PSY实现了跨越式提升：

| 时期 | 行业平均PSY（头） | 头部企业PSY（头） | 技术驱动因素 |
|------|------------------|------------------|-------------|
| 2010-2014 | 16-18 | 20-22 | 引进外来品种（丹系、法系、美系），人工授精普及 |
| 2015-2018 | 18-20 | 22-24 | 批次化生产管理，母猪营养配方优化 |
| 2019-2021 | 19-21 | 24-26 | 非瘟后种群优化，低效母猪淘汰，智能化环控 |
| 2022-2024 | 22-24 | 26-28 | 基因组选择育种，精准饲喂，疫病净化 |
| 2025-2026 | 24-25 | 28-30 | 空气过滤系统，智能环控，种猪自主育种体系成熟 |

2025-2026年，行业平均PSY达到24-25头，较2010年代的18头提升了33%-39%。头部企业表现更为突出：牧原股份PSY达到28-30头，温氏股份26-28头，神农集团28头以上。这意味着，同等能繁母猪存栏水平下，行业提供的仔猪数量和最终的生猪出栏量，较十年前增加了约三分之一。

#### 4.2.2 "隐形产能"效应的量化分析

PSY提升形成了巨大的"隐形产能"，使得能繁母猪存栏指标对实际供给的指示意义大幅下降。我们可以通过标准化折算来量化这一效应：

| 时期 | 能繁母猪存栏（万头） | PSY（头） | 标准化产能（以PSY=18为基准，万头） |
|------|----------------------|----------|-----------------------------------|
| 2014年Q1 | 4390 | 18 | 4390 |
| 2018年Q1 | 3360 | 19 | 3547 |
| 2022年Q1 | 4177 | 22 | 5105 |
| 2024年Q1 | 4042 | 24 | 5389 |
| 2026年Q1 | 3904 | 25 | 5422 |

2026年Q1的3904万头能繁母猪，按PSY=25计算理论年仔猪产出约9.76亿头；折算为PSY=18时代的当量，则相当于5422万头能繁母猪的产出能力，实际产能供给能力较2014年的4390万头高出约23.5%。这一"隐形产能"效应解释了为何当前能繁母猪存栏已较2024年高点去化3.4%，但猪价仍未企稳回升——因为实际产能供给并未真正收缩，甚至可能仍在扩张。

#### 4.2.3 能繁母猪存栏指标的参考价值折损

PSY的跃升导致能繁母猪存栏作为周期领先指标的预测效力显著下降。我们可以通过历史回测来验证这一结论：

| 预测模型 | 历史样本期 | R²（拟合优度） | 预测误差（MAPE） |
|----------|-----------|---------------|-----------------|
| 模型A：能繁母猪存栏（名义值）→10个月后猪价 | 2010-2018 | 0.52 | 18.5% |
| 模型A：能繁母猪存栏（名义值）→10个月后猪价 | 2019-2026 | 0.18 | 31.2% |
| 模型B：标准化产能（PSY调整）→10个月后猪价 | 2019-2026 | 0.38 | 24.6% |
| 模型C：能繁母猪存栏+仔猪价格+猪粮比→猪价 | 2019-2026 | 0.45 | 21.8% |

回测结果显示，单纯依赖能繁母猪存栏名义值预测猪价的模型，在2019-2026年样本期的R²从0.52骤降至0.18，预测误差（MAPE）从18.5%扩大至31.2%。即便引入PSY调整的标准化产能，预测效力也仅能部分恢复（R²=0.38）。这说明，除PSY外，规模化行为、政策干预等因素也在削弱传统指标的预测能力。

### 4.3 变化三：政策调控常态化——"托底不抬顶"的新范式

#### 4.3.1 产能调控机制的精细化

2021年以来，农业农村部逐步建立了以能繁母猪存栏为核心的产能调控机制，标志着生猪产业政策从"应急干预"转向"预期管理"。

| 政策节点 | 核心内容 | 调控目标 |
|----------|----------|----------|
| 2021年9月 | 《生猪产能调控实施方案（暂行）》 | 设定能繁母猪正常保有量4100万头，上下浮动5%为绿色区间 |
| 2023年3月 | 修订产能调控方案 | 正常保有量从4100万头下调至3900万头 |
| 2024年8月 | 再次修订产能调控方案 | 正常保有量从3900万头下调至3750万头；细化绿色、黄色、红色预警区间 |
| 2025年至今 | 常态化监测预警 | 年出栏10万头以上集团纳入重点监测；月度数据发布 |

能繁母猪存栏预警区间的量化标准：

| 预警级别 | 能繁母猪存栏区间（万头） | 对应存栏变动 | 政策响应 |
|----------|------------------------|-------------|----------|
| 绿色区间 | 3563-3938（3750±5%） | 正常波动 | 市场自发调节，不干预 |
| 黄色区间 | 3375-3563 或 3938-4125（±5%-10%） | 轻度偏离 | 启动引导，发布预警信息 |
| 红色区间 | <3375 或 >4125（偏离>10%） | 严重偏离 | 强制干预，启动收储/投放 |

2026年Q1能繁母猪存栏3904万头，处于绿色区间上沿（3938万头）附近，尚未触发黄色预警。这意味着，尽管猪价已创近十年新低，但从政策角度看，产能水平仍处于"合理区间"，不需要强力干预。这一政策框架的存在，本身就对养殖户的预期产生了深远影响——大家知道政府会在极端情况下托底，但不会在轻度亏损期介入，从而加速了"亏损耐受"行为的形成。

#### 4.3.2 储备肉收储与投放的常态化

中央储备冻猪肉的收储和投放机制，在2022年后进入常态化运行阶段：

| 时间 | 操作 | 数量（万吨） | 猪价影响 |
|------|------|-------------|----------|
| 2022年3-6月 | 收储 | 13批次，约10万吨 | 短期提振0.3-0.5元/公斤 |
| 2022年9-11月 | 投放 | 6批次，约5万吨 | 平抑涨幅0.2-0.4元/公斤 |
| 2023年2-5月 | 收储 | 8批次，约6万吨 | 短期提振0.2-0.3元/公斤 |
| 2024年3-6月 | 收储 | 10批次，约8万吨 | 短期提振0.2-0.4元/公斤 |
| 2025年2-4月 | 收储 | 6批次，约5万吨 | 短期提振0.1-0.3元/公斤 |
| 2025年9-11月 | 投放 | 4批次，约3万吨 | 平抑涨幅0.1-0.2元/公斤 |
| 2026年3-5月 | 收储 | 4批次，约3万吨 | 提振效果微弱（<0.1元/公斤） |

收储/投放的常态化运行，产生了两个显著效果：一是"托底"——在猪价极端低迷时通过收储向市场释放信号，防止恐慌性抛售；二是"不抬顶"——在猪价上涨时通过投放增加供给，防止价格过度上涨。这种"双向调节"机制有效压缩了猪价的波动区间，将周期振幅从历史上的2-3倍收窄至当前的1.5倍以内。

值得注意的是，2026年3-5月的4批次收储对猪价的提振效果已极为微弱（<0.1元/公斤），表明在市场供给宽松的大背景下，收储的政策边际效应正在递减。这进一步印证了当前周期的核心矛盾——不是需求不足或政策缺位，而是供给端的结构性过剩。

#### 4.3.3 进口肉的动态调节

进口猪肉是调节国内供需平衡的另一重要工具。中国猪肉进口量在2020年达到439万吨的历史峰值（占国内消费量约8.5%），此后随着国内产能恢复逐步回落：

| 年份 | 猪肉进口量（万吨） | 占国内消费量比例 | 政策导向 |
|------|-------------------|-----------------|----------|
| 2019 | 211 | 4.0% | 增加进口以弥补非瘟缺口 |
| 2020 | 439 | 8.5% | 历史峰值，大量进口平抑高价 |
| 2021 | 371 | 7.0% | 维持高位进口 |
| 2022 | 286 | 5.3% | 随国内产能恢复逐步减少 |
| 2023 | 220 | 4.0% | 正常化进口 |
| 2024 | 180 | 3.2% | 控制进口以保护国内养殖户 |
| 2025 | 150 | 2.6% | 低位运行，动态调节 |
| 2026年（预测） | 130-140 | 2.2%-2.4% | 视国内价格走势灵活调整 |

2025-2026年，猪肉进口量控制在150万吨左右的低位，占国内消费量比例降至2.6%以下。这一政策导向体现了"保护国内产能"的意图——在猪价低迷期减少进口，为国内养殖户留出市场空间；若未来猪价大幅上涨，则可迅速增加进口以平抑价格。

#### 4.3.4 头部企业纳入重点监测的影响

2024年修订的《生猪产能调控实施方案》明确将年出栏10万头以上的养殖集团纳入农业农村部重点监测范围。这意味着，牧原（7798万头）、温氏（4048万头）、双胞胎（2643万头）等头部企业的生产计划、存栏变动、出栏节奏，均处于监管部门的直接监测之下。

这一政策的深远影响在于：**头部企业的产能扩张行为不再是纯粹的市场决策，而是受到政策预期的约束**。当能繁母猪存栏接近绿色区间上沿（3938万头）时，监管部门可通过窗口指导、预警约谈等方式，引导头部企业放缓扩张节奏；当存栏接近下沿（3563万头）时，则可鼓励企业稳定或适度增加产能。这种"预期管理"机制，进一步平滑了周期的波动。

### 4.4 三因子交互作用的综合效应

PSY效率因子、规模化因子和政策调控因子的交互作用，共同塑造了2026年猪周期的新特征：

| 因子 | 单独效应 | 交互效应 |
|------|----------|----------|
| PSY效率因子 | 同等母猪存栏下供给增加33%-67% | 与规模化叠加：头部企业PSY更高，产能集中度提升速度超预期 |
| 规模化因子 | 亏损期去产能速度放缓50%-70% | 与政策叠加：头部企业受政策监测，产能行为更具可预测性 |
| 政策调控因子 | 周期振幅收窄30%-50% | 与效率因子叠加：政策以名义母猪存栏为锚，但实际产能被高估 |

三因子的综合效应可以概括为：**供给弹性下降、需求弹性不变→价格波动收窄→周期弱化**。传统周期中猪价下跌10%会触发5%-8%产能去化，当前周期中猪价下跌30%仅触发3%-5%去化，供给弹性系数从-0.5降至-0.1。价格波动幅度显著减小：2022-2026年猪价振幅1.95倍，而2018-2022年振幅3.8倍。周期底部时长从6-12个月延长至16个月以上。盈利中枢下移至12-18元/公斤，暴利期（猪价>20元/公斤）将极为罕见。

## 五、2026年下半年猪价走势预判

基于前述分析框架，我们对2026年下半年猪价走势进行分季度预判。需要强调的是，在新的周期范式下，猪价的波动幅度和上涨速度都将显著弱于历史周期，投资者应调整对"周期反转"力度和节奏的预期。

### 5.1 Q3（7-9月）：磨底尾声，低位震荡

#### 5.1.1 供需格局分析

2026年Q3的生猪出栏量，主要由2025年Q4至2026年Q1的能繁母猪存栏决定。根据前文数据，2025年Q4能繁母猪存栏3915万头，2026年Q1为3904万头，处于高位运行状态。按PSY=25、育肥成活率92%计算，Q3理论出栏量约为1.75-1.80亿头，同比仅下降约1.0%-1.5%。

需求端，Q3处于传统消费淡季（暑期学校放假、高温抑制肉类消费），但存在中秋备货（9月中旬）的短期拉动。预计Q3总消费量约为1.72-1.78亿头，供需基本平衡或轻微过剩（过剩幅度约1%-2%）。

#### 5.1.2 价格区间预判

Q3均价预测9.2-10.0元/公斤：7月9.2-9.8元/公斤（高温淡季，出栏压力大），8月9.0-9.6元/公斤（消费最低迷，可能创年内新低），9月9.5-10.5元/公斤（中秋国庆备货启动）。8月可能创全年最低点，但下探空间有限——猪粮比已接近3.8:1历史低位，继续下跌将触发更强力收储政策，且养殖端会出现抗价惜售行为。

#### 5.1.3 关键风险点

- **极端天气风险**：南方高温干旱或洪涝灾害可能影响生猪调运，短期推高区域价格。
- **疫病风险**：夏季高温高湿易诱发猪病，若出现区域性疫情可能加速局部产能去化。
- **二次育肥扰动**：若养殖户基于"四季度旺季"预期在Q3进行二次育肥，可能短期减少标猪供给、推高价格，但会在Q4集中释放，加大价格压力。

### 5.2 Q4（10-12月）：温和回升，腌腊季需求拉动

#### 5.2.1 供需格局分析

Q4理论出栏量约为1.78-1.82亿头，同比下降约2.0%-3.0%，为2025年以来首次出现出栏量同比下降，供给压力的边际缓解为猪价回升创造条件。需求端，Q4是传统消费旺季：10月国庆长假聚餐消费，11月南方腌腊启动拉动鲜猪肉需求，12月冬至元旦双节备货达到年内峰值。预计Q4总消费量约为1.85-1.90亿头，供需关系从轻微过剩转向轻微紧缺（紧缺幅度约2%-3%）。

#### 5.2.2 价格区间预判

Q4均价预测10.8-11.8元/公斤：10月10.0-11.0元/公斤（国庆需求拉动），11月11.0-12.0元/公斤（腌腊季启动），12月11.5-12.5元/公斤（冬至元旦备货，年内高点）。12月可能触及全年最高点，但仍低于行业平均完全成本13.0元/公斤，行业整体仍处于亏损状态。

#### 5.2.3 涨幅受限的核心原因

1. **能繁母猪存栏仍高于合理水平**：2026年末预计仍在3820-3850万头，高于3750万头正常保有量，2027年Q2之前生猪出栏量仍将维持高位。
2. **储备肉投放的压制**：若Q4猪价突破13元/公斤，政府将启动储备肉投放。每投放1万吨可短期压制猪价约0.1-0.2元/公斤。
3. **二次育肥的"蓄水池"效应**：若Q3末-Q4初二次育肥集中，大体重猪只将在2027年Q1集中出栏，加大节后供给压力。
4. **消费需求结构性疲软**：受宏观经济环境影响，2026年猪肉消费总量预计较2024年下降约2%-3%，需求增量有限。

### 5.3 关键拐点：2026年9月前后

综合上述分析，2026年9月前后是观察猪周期拐点的关键时间窗口：

9月前后观察以下关键信号：供给端（生猪出栏量同比由正转负）、需求端（屠宰量环比增幅>10%）、价格端（猪价连续2周环比上涨突破10元/公斤）、政策端（收储批次增加单次规模扩大）、情绪端（出栏均重上升标猪供应减少）。若上述信号同步出现，则可确认猪价脱离底部区域进入温和回升通道。但需注意，此"拐点"仅是"磨底结束、震荡回升"的信号，而非"V型反转、暴利归来"的信号。在新周期范式下，价格回升将是缓慢、曲折的过程。

### 5.4 全年预判：前低后高，年度涨幅温和

2026年全年猪价呈现"前低后高"走势：Q1均价14.5元/公斤（盈亏平衡至微利），Q2均价10.5元/公斤（深度亏损，环比-27.6%），Q3均价9.6元/公斤（重度亏损），Q4均价11.3元/公斤（轻度亏损，环比+17.7%）。全年均价11.5元/公斤，较2025年均价13.2元/公斤下跌约12.9%。年末（12月）猪价预测11.5-12.5元/公斤，较年初14.5元/公斤下跌约14%-21%。另有部分机构给出年末16-17元/公斤的乐观预测，但实现概率较低（<30%）。

### 5.5 与历史周期的对比：涨幅显著收窄

| 周期 | 底部价格（元/公斤） | 回升期高点（元/公斤） | 涨幅 | 回升时长 |
|------|-------------------|---------------------|------|----------|
| 2010年周期 | 9.0 | 19.9（2011年9月） | +121% | 17个月 |
| 2014年周期 | 10.5 | 21.0（2016年6月） | +100% | 26个月 |
| 2018年周期 | 10.0 | 38.0（2020年2月） | +280% | 22个月 |
| 2022年周期 | 12.0 | 28.0（2022年10月） | +133% | 7个月 |
| 2026年周期（预测） | 9.5 | 12.5（2026年12月） | +32% | 4-5个月 |

从对比中可以清晰看出，2026年周期的回升幅度（预测+32%）和回升时长（4-5个月）均远小于历史周期。这并非偶然，而是三因子（PSY效率×规模化×政策调控）共同作用下的结构性结果。未来猪周期的运行特征，将越来越趋近于"窄幅波动、弱周期、温和上涨/下跌"的新常态。

## 六、产业链量化分析

生猪养殖产业链涵盖上游（饲料原料：玉米、豆粕）、中游（生猪养殖）和下游（屠宰加工、零售消费）三大环节。在猪价深度磨底的背景下，产业链各环节的利润分配格局发生显著变化，成本传导机制和价格弹性也呈现出新的特征。

### 6.1 上游：饲料原料的成本压力与价格走势

#### 6.1.1 饲料成本结构分析

饲料成本是生猪养殖的最大成本项，占完全成本的65%-70%。配合饲料主要由玉米（能量原料，占比约55%-60%）、豆粕（蛋白原料，占比约20%-25%）、麸皮/鱼粉/氨基酸等（占比约15%-20%）构成。

| 饲料原料 | 在配合饲料中占比 | 2026年6月价格（元/吨） | 同比变化 | 在养殖成本中贡献（元/公斤） |
|----------|-----------------|----------------------|----------|---------------------------|
| 玉米 | 55%-60% | 2420 | -2.0% | 4.8-5.2 |
| 豆粕 | 20%-25% | 3250 | -8.0% | 2.0-2.4 |
| 麸皮/其他 | 15%-20% | 1800-2200 | -3.0% | 1.2-1.5 |
| 配合饲料综合 | 100% | 约3100 | -4.5% | 8.0-9.1 |

2026年上半年，饲料原料价格整体呈下行态势，主要得益于全球粮食供应链的恢复和国内玉米、大豆进口增加。但下半年饲料原料价格面临上行压力：

- **玉米**：2026年全球玉米主产区（美国、巴西、乌克兰）遭遇极端天气（厄尔尼诺转拉尼娜），产量预期下调。美国农业部（USDA）7月报告将2026/27年度全球玉米产量预测下调约3%，芝加哥期货交易所（CBOT）玉米期货价格较年初上涨约8%。预计国内玉米价格Q3-Q4将从2420元/吨回升至2550-2650元/吨，涨幅5%-9%。

- **豆粕**：2026年下半年南美大豆（巴西、阿根廷）出口旺季过后，全球大豆供应趋紧。同时，国内生猪存栏虽在去化，但绝对量仍处高位，豆粕需求保持刚性。预计豆粕价格Q3-Q4将从3250元/吨回升至3400-3600元/吨，涨幅5%-11%。

#### 6.1.2 饲料成本对养殖盈利的敏感性分析

饲料价格波动对养殖头均盈利影响显著。以出栏体重120公斤、料肉比2.8:1计算，配合饲料价格每上涨200元/吨，头均亏损扩大约216元。若2026年下半年饲料价格上涨10%-15%（从3100元/吨涨至3400-3500元/吨），而猪价仅回升15%-20%，则养殖企业盈利修复速度将慢于猪价回升速度，行业扭亏为盈时间可能推迟至2027年Q2以后。

#### 6.1.3 头部企业的饲料成本优势

头部企业凭借规模化采购、自有饲料加工、配方优化等手段，建立了显著的饲料成本优势：

| 企业 | 饲料自给率 | 玉米采购价较市场低（元/吨） | 豆粕采购价较市场低（元/吨） | 配方技术优势 |
|------|-----------|---------------------------|---------------------------|-------------|
| 牧原股份 | 100% | -80至-120 | -100至-150 | 低蛋白日粮技术，豆粕用量降低3-5pct |
| 温氏股份 | 95% | -60至-100 | -80至-120 | 精准营养配方，饲料转化率行业领先 |
| 新希望 | 90% | -50至-80 | -60至-100 | 多元化蛋白源替代（菜粕、棉粕） |
| 海大集团 | 100% | -40至-60 | -50至-80 | 水产饲料技术迁移，适口性优化 |
| 行业平均 | 30% | 0 | 0 | 标准配方 |

以牧原股份为例，其自有饲料加工厂遍布全国主要养殖区域，玉米采购通过直接从东北产区集港运输，省去中间贸易商环节；豆粕采购与大型油厂签订长期协议，锁定基差。综合下来，牧原的饲料成本较行业平均低约0.4-0.6元/公斤，对应头均成本优势约50-70元。

### 6.2 中游：养殖环节的成本梯队分化

#### 6.2.1 完全成本的量化拆解

生猪养殖的完全成本包括饲料成本、仔猪成本、人工成本、折旧与财务费用、兽药疫苗费用、水电燃料费用等。以行业平均完全成本13.0元/公斤为例，成本结构如下：

| 成本项目 | 金额（元/公斤） | 占比 |
|----------|----------------|------|
| 饲料成本 | 8.5-9.1 | 65%-70% |
| 人工成本 | 1.0-1.3 | 8%-10% |
| 折旧与财务费用 | 1.3-1.6 | 10%-12% |
| 兽药疫苗 | 0.65-0.78 | 5%-6% |
| 水电燃料 | 0.3-0.4 | 2%-3% |
| 其他费用 | 0.5-0.7 | 4%-5% |
| **完全成本合计** | **13.0** | **100%** |



#### 6.2.2 成本梯队的量化对比

2026年Q2，不同养殖主体的完全成本呈现显著分化：

| 梯队 | 代表企业/群体 | 完全成本（元/公斤） | 较行业平均差异 | 成本优势来源 |
|------|-------------|--------------------|---------------|-------------|
| 第一梯队 | 牧原股份 | 11.6 | -10.8% | 自繁自养、饲料自给、智能化管理、规模效应 |
| 第二梯队 | 温氏股份、神农集团 | 12.0-12.3 | -5.4%至-7.7% | "公司+农户"模式轻资产、区域优势 |
| 第三梯队 | 海大集团、双胞胎 | 12.3-12.8 | -1.5%至-5.4% | 饲料主业协同、成本管控严格 |
| 第四梯队 | 新希望、天邦食品 | 12.8-13.2 | -1.5%至+1.5% | 多元化布局，养殖效率中等 |
| 第五梯队 | 正邦科技（整合期） | 13.0-13.5 | 0%至+3.8% | 整合期管理动荡，效率待恢复 |
| 第六梯队 | 中型规模场 | 13.0-14.0 | 0%至+7.7% | 规模不经济，管理水平参差 |
| 第七梯队 | 散户 | 14.0-15.5 | +7.7%至+19.2% | 饲料采购价高、人工效率低、疫病损失大 |

成本分化的直接后果是：**在深度亏损周期中，低成本企业凭借"成本壁垒"维持现金流安全和市场份额，高成本企业被迫加速退出，行业集中度进一步提升**。2026年Q1的数据显示，牧原股份在头均亏损366元的情况下，凭借经营性现金流和融资储备，仍维持正常运营和适度扩张；而散户在头均亏损500元以上的压力下，加速退出市场。

#### 6.2.3 自繁自养 vs 外购育肥模式对比

| 对比维度 | 自繁自养模式（牧原为代表） | 外购育肥模式（部分中小企业） |
|----------|--------------------------|---------------------------|
| 完全成本 | 11.6元/公斤（2026年Q2） | 14.0-15.0元/公斤 |
| 成本稳定性 | 高（不受仔猪价格波动影响） | 低（仔猪价格波动大） |
| 资产投入 | 重资产（种猪、母猪场、育肥场） | 轻资产（仅需育肥场） |
| 疫病风险 | 封闭体系，生物安全可控 | 外购仔猪来源复杂，疫病风险高 |
| 周期适应性 | 弱周期中抗风险能力强 | 弱周期中现金流压力大 |
| 盈利弹性 | 猪价上涨时利润放大效应强 | 猪价上涨时利润放大效应弱（仔猪成本同步上涨） |

自繁自养模式在弱周期中的优势尤为明显。以2026年Q2为例，自繁自养头均亏损约366元（牧原），而外购育肥头均亏损可能达到700-900元（因15公斤仔猪价格约350-400元/头，折算育肥成本约3.0-3.5元/公斤）。这一巨大的成本差异，解释了为何牧原在深度亏损期仍能维持扩张，而外购育肥为主的中小企业则大量退出。

### 6.3 下游：屠宰加工与零售消费

#### 6.3.1 屠宰环节的利润空间

屠宰加工环节的利润主要取决于"屠宰价差"（白条肉批发价 - 生猪收购价）和"头均屠宰量"。

| 指标 | 2026年Q2数据 | 同比变化 |
|------|-------------|----------|
| 生猪收购价（元/公斤） | 9.5-9.8 | -18.5% |
| 白条肉批发价（元/公斤） | 12.5-13.0 | -16.0% |
| 屠宰价差（元/公斤） | 3.0-3.2 | +12.0% |
| 头均屠宰利润（元/头） | 40-60 | -20.0% |

2026年上半年，屠宰企业面临"两头受压"困境：上游生猪收购价虽低，但下游白条肉批发价同样低迷，且终端消费疲软导致屠宰量不足、产能利用率下降。大型屠宰企业产能利用率约55%-65%，较正常年份的75%-85%明显下降。屠宰价差（3.0-3.2元/公斤）较历史均值（4.0元/公斤）收窄，表明下游零售端价格传导效率下降，终端零售商通过压缩自身利润维持销量，导致屠宰企业议价空间被挤压。

#### 6.3.2 零售端价格与替代品效应

| 猪肉品类 | 2026年6月零售价（元/公斤） | 同比变化 |
|----------|--------------------------|----------|
| 五花肉 | 16.0-18.0 | -15.0% |
| 里脊肉 | 20.0-22.0 | -12.0% |
| 排骨 | 22.0-25.0 | -10.0% |
| 肉馅（统货） | 14.0-15.0 | -18.0% |

### 6.4 核心效率指标的量化对比

#### 6.4.1 猪粮比

猪粮比是衡量养殖盈利水平和政策干预阈值的核心指标。2026年Q2猪粮比约为4.0:1，处于重度亏损区间。

猪粮比长期低于5.5:1的收储预警线，但产能去化并未如预期加速，这是当前周期与传统周期最显著的差异之一。其背后的结构性原因，正是前文分析的规模化企业抗亏损能力和政策托底效应。

#### 6.4.2 料肉比

料肉比（每增重1公斤消耗的饲料量）是衡量养殖效率的关键指标。料肉比越低，饲料成本占比越小，养殖效率越高。

| 养殖主体 | 料肉比 | 较行业平均差异 | 技术来源 |
|----------|--------|---------------|----------|
| 牧原股份 | 2.6-2.7 | -7.1%至-10.7% | 精准饲喂、品种优化、环境控制 |
| 温氏股份 | 2.7-2.8 | -3.6%至-7.1% | "公司+农户"模式下的统一饲料配方 |
| 行业平均 | 2.8-2.9 | 基准 | — |
| 散户 | 3.0-3.3 | +3.6%至+13.8% | 饲料浪费、品种混杂、疫病影响 |

料肉比的差异，在饲料价格高企时被进一步放大。以饲料价格3.1元/公斤计算，料肉比2.6与3.3之间的成本差距为：

**(3.3 - 2.6) × 3.1元/公斤 = 2.17元/公斤**

这意味着，仅料肉比一项，散户就较牧原股份多承担约2.2元/公斤的成本，对应头均成本劣势约260元。这一效率差距，在深度亏损周期中成为决定企业生死的关键变量。

#### 6.4.3 养殖利润率的周期波动

从养殖利润率的周期波动可以看出，2024-2026年的亏损深度和持续时间均超过历史同期，但产能去化速度却更慢。这一"深度亏损、缓慢去化"的新特征，正是规模化主导和效率提升双重作用下的产物。

## 八、投资策略与交易窗口

### 8.1 短期策略（1-3个月）：左侧布局窗口，产能去化加速期

#### 8.1.1 时间窗口：2026年7月-9月

2026年Q3是猪周期左侧布局的关键窗口期。从历史规律看，猪价在8-9月创年内新低后，随着中秋国庆备货启动和供给同比转负，价格有望企稳回升。对于长期投资者而言，在猪价最悲观、市场情绪最低迷时布局，往往能获得最佳的周期回报。

| 指标 | 当前状态（2026年6月） | 左侧布局信号 |
|------|---------------------|-------------|
| 猪价 | 9.5-9.8元/公斤，创近十年新低 | 价格极度悲观，下行空间有限 |
| 猪粮比 | 4.0:1，重度亏损 | 政策托底预期增强 |
| 能繁母猪去化 | 连续9个月环比下降，累计3.5% | 去化趋势确立，虽缓慢但方向明确 |
| 市场情绪 | 极度悲观，养殖板块估值低位 | 逆向布局的时机 |
| 养殖板块PB | 头部企业1.5-2.5倍 | 历史底部区间 |

#### 8.1.2 标的选择与仓位管理

短期左侧布局应聚焦于**成本最低、资金最充裕的龙头企业**，以确保在周期底部延长的情况下仍能 survivable。

| 标的 | 建议仓位 | 入场区间 | 止损位 | 目标位 |
|------|----------|----------|--------|--------|
| 牧原股份 | 30%-40% | 35-45元 | 30元（-20%） | 60-70元（+50%） |
| 温氏股份 | 20%-30% | 15-20元 | 13元（-20%） | 25-30元（+50%） |
| 海大集团 | 20%-30% | 40-50元 | 35元（-20%） | 60-70元（+30%） |

#### 8.1.3 关键观测指标

- **能繁母猪存栏**：若2026年Q2-Q3能繁母猪去化速度加快至月均0.5%-0.8%，则确认去化加速，左侧布局信号增强。
- **猪粮比**：若跌破3.8:1并触发更强力的收储政策，则政策底出现，市场底不远。
- **仔猪价格**：若15公斤仔猪价格跌破300元/头，表明补栏意愿极度低迷，预示未来供给收缩。
- **出栏均重**：若出栏均重从当前的125-130公斤降至115-120公斤，表明养殖户在加速出栏、减少压栏，有利于缩短供给高峰。

### 8.2 中期策略（3-6个月）：Q4旺季反弹，目标价区间

#### 8.2.1 时间窗口：2026年10月-12月

Q4是传统消费旺季，腌腊、冬至、元旦等节日备货将拉动需求环比提升15%-20%。若供给端在Q4首次出现同比下降（预计同比-2%至-3%），则供需关系将从轻微过剩转向轻微紧缺，猪价有望温和回升至11.0-12.5元/公斤。

| 月份 | 猪价预测（元/公斤） | 对应养殖企业状态 | 板块情绪 |
|------|-------------------|----------------|----------|
| 10月 | 10.0-11.0 | 大部分企业仍亏损，牧原微利 | 悲观修复 |
| 11月 | 11.0-12.0 | 头部企业接近盈亏平衡 | 谨慎乐观 |
| 12月 | 11.5-12.5 | 牧原、温氏实现微利 | 乐观 |

#### 8.2.2 目标价区间

基于Q4猪价预测，中期目标价区间：牧原股份55-65元（+38%-63%），温氏股份25-30元（+39%-67%），海大集团55-65元（+22%-44%），新希望14-16元（+17%-33%）。上述目标价基于猪价回升至11.5-12.5元/公斤的假设，若回升不及预期则下调20%-30%。

#### 8.2.3 交易策略

中期策略以"持有+波段"为主：Q3左侧布局仓位在Q4猪价回升过程中继续持有；若11月猪价快速上涨至12.5元/公斤以上，可适当减仓30%-50%锁定利润（节后存在二次育肥出栏回调风险）；若Q4猪价提前下跌至10.5元/公斤以下，则逢低加仓等待2027年Q2拐点。

### 8.3 长期策略（6-12个月）：2027H1供给收缩，猪价上行周期

#### 8.3.1 周期拐点的确认条件

2027年上半年是观察本轮周期是否真正反转的关键窗口。若以下条件同时满足，则可确认进入新一轮猪价上行周期：

| 条件 | 量化标准 | 预计实现时间 |
|------|----------|-------------|
| 能繁母猪存栏降至合理水平 | <3750万头（正常保有量） | 2027年Q1-Q2 |
| 猪粮比回到盈亏平衡线 | >6.0:1 | 2027年Q2-Q3 |
| 生猪出栏量同比下降 | 同比-3%至-5% | 2027年Q1 |
| 行业养殖利润转正 | 自繁自养头均盈利>200元 | 2027年Q2 |
| 头部企业出栏增速放缓 | 前10家出栏同比<3% | 2027年Q1 |

#### 8.3.2 2027年猪价预测

若上述条件在2027年Q1-Q2满足，则2027年下半年猪价有望进入上行通道：

| 季度 | 猪价预测（元/公斤） | 驱动因素 |
|------|-------------------|----------|
| 2027年Q1 | 10.5-11.5 | 节后消费回落，二次育肥出栏压力 |
| 2027年Q2 | 11.0-13.0 | 能繁母猪去化传导至出栏，供给收缩 |
| 2027年Q3 | 13.0-16.0 | 供给缺口显现，旺季需求拉动 |
| 2027年Q4 | 14.0-17.0 | 腌腊季+供给持续偏紧 |

2027年全年均价预测：13.0-15.0元/公斤，较2026年的11.5元/公斤上涨13%-30%。若出现极端供给收缩（能繁母猪降至3600万头以下），则2027年下半年猪价可能突破18-20元/公斤，但这一情景的实现概率较低（约20%）。

#### 8.3.3 长期仓位管理

长期投资者可在2026年Q3-Q4建立底仓（目标仓位50%-70%），在2027年Q1确认拐点后加仓至满仓。若2027年Q1猪价因二次育肥出栏而回调，则回调即是加仓良机。

### 8.4 适配量化策略

#### 8.4.1 周期底部反转策略

核心逻辑：在行业极度悲观时买入低成本龙头，等待周期回暖。选股条件为完全成本<12.5元/公斤、资产负债率<65%、出栏规模>1000万头。买入信号：猪粮比<4.5:1且能繁母猪存栏连续3个月环比下降。卖出信号：猪粮比>8.0:1且自繁自养头均盈利>800元。仓位管理：初始仓位30%，每下跌10%加仓10%，最大仓位70%。止损规则：个股跌幅达-25%或猪粮比跌破3.5:1超过3个月。

#### 8.4.2 行业轮动策略

根据周期位置动态调整配置比例：周期底部配置养殖板块40%-50%、饲料30%、种业10%；周期回升期养殖60%-70%、饲料20%、动保10%；周期高峰期养殖30%-40%、屠宰加工30%、食品20%；周期下跌期养殖10%-20%、饲料40%、种业20%。

#### 8.4.3 配对交易策略

适用场景：对周期方向不确定但认为龙头企业将跑赢行业。典型配对：做多牧原股份/做空新希望（低成本龙头vs高成本二线），做多温氏股份/做空正邦科技（稳健模式vs重组不确定性），做多海大集团/做空天邦食品（饲料主业支撑vs纯养殖承压）。

### 8.5 仓位建议与止盈止损规则

#### 8.5.1 仓位建议

保守型（低风险偏好）：养殖板块仓位10%-15%，核心持仓牧原40%+温氏30%+海大30%。稳健型（中低风险偏好）：仓位15%-25%，核心持仓牧原35%+温氏25%+海大25%，卫星持仓益生15%。平衡型（中风险偏好）：仓位25%-35%，核心持仓牧原30%+温氏25%+海大20%，卫星持仓益生15%+新五丰10%。进取型（高风险偏好）：仓位35%-50%，核心持仓牧原30%+温氏20%+海大15%，卫星持仓益生15%+新希望10%+正邦10%。

#### 8.5.2 止盈规则

猪粮比>8.0:1且持续2个月、自繁自养头均盈利>1000元、目标价达成（牧原65元/温氏30元/海大65元）或板块PB>3.5倍时，减仓30%-50%锁定利润。

#### 8.5.3 止损规则

个股跌幅达-25%、猪粮比跌破3.5:1持续3个月、能繁母猪存栏连续2个月环比回升、头部企业出栏同比增幅>10%或重大疫病暴发时，减仓至50%以下或清仓观望。

## 十、核心结论与前瞻

### 10.1 新周期特征：窄幅波动、弱周期、温和上涨

2026年猪周期的"变天"，并非周期消失，而是周期特征的结构性转变。在PSY效率跃升、规模化主导和政策调控常态化三大因子的共同作用下，未来猪周期将呈现出以下新特征：

| 维度 | 传统周期（2006-2022） | 新周期（2023-） | 变化 |
|------|----------------------|----------------|------|
| 周期时长 | 36-48个月 | 48-60个月 | 延长25%-50% |
| 价格振幅（峰值/谷底） | 2.0-3.8倍 | 1.5-2.0倍 | 收窄40%-50% |
| 底部持续时间 | 6-12个月 | 12-24个月 | 延长1-2倍 |
| 去化幅度（能繁母猪） | 10%-16% | 5%-10% | 下降40%-60% |
| 暴利期（猪价>20元/公斤） | 12-18个月 | 罕见或极短（<6个月） | 基本消失 |
| 盈利中枢 | 13-18元/公斤 | 12-16元/公斤 | 下移10%-15% |
| 集中度提升速度 | 年均+2pct | 年均+4pct | 加速1倍 |

新周期的核心特征是**"窄幅波动、弱周期、温和上涨/下跌"**。猪价将围绕成本线（12-14元/公斤）在较窄区间内波动，极端高价（>25元/公斤）和极端低价（<8元/公斤）的出现概率大幅降低。这对于消费者而言是利好（猪肉价格稳定亲民），但对于养殖企业和投资者而言，意味着**周期投资的难度显著上升、回报预期需要下调**。

### 10.2 行业格局：集中度持续提升，头部企业市占率将超50%

在新周期范式下，行业集中度提升速度将进一步加快。低成本龙头企业凭借"成本壁垒+资金壁垒+技术壁垒"，将在每一轮周期底部扩大市场份额；高成本散户和中小规模场则在亏损周期中加速退出。预计到2030年，前10家上市猪企合计市占率将达50%，行业进入"寡头竞争"阶段。寡头竞争格局下，头部企业产能博弈将更加理性，行业供给稳定性和价格可预测性将提升。

| 年份 | 前5家市占率 | 前10家市占率 | 前20家市占率 | 规模化率（>500头） |
|------|------------|-------------|-------------|-------------------|
| 2025 | 25% | 33% | 36% | 70% |
| 2027（预测） | 32% | 42% | 48% | 78% |
| 2030（预测） | 40% | 50% | 58% | 85% |

### 10.3 投资主线：低成本龙头优先受益，高成本企业面临淘汰

在新周期范式下，养殖板块的投资逻辑发生根本性转变：**从"博弈周期反转"转向"选择低成本龙头"**。传统周期投资核心是判断拐点 timing，在底部买入、顶部卖出；新周期中拐点 timing 难以精准判断，且上涨幅度有限，投资胜负手不再是"何时买"，而是"买谁"。

| 投资逻辑 | 传统周期 | 新周期 |
|----------|----------|--------|
| 核心策略 | 周期底部博反弹，周期顶部逃顶 | 持有低成本龙头，穿越周期 |
| 选股标准 | 弹性最大（高杠杆、高成本） | 成本最低、资金最充裕 |
| 持有周期 | 6-18个月 | 3-5年 |
| 预期回报 | 周期高点翻倍或更高 | 年均15%-25% |
| 风险特征 | 高波动、高回撤 | 低波动、稳健增长 |

低成本龙头企业的竞争优势在新周期中被放大：**牧原股份**完全成本11.6元/公斤，在猪价12-16元/公斤的新盈利中枢下可实现持续盈利；**温氏股份**"公司+农户"模式的轻资产优势在资本开支收紧环境下更具吸引力；**海大集团**饲料主业的稳定现金流为养殖板块提供安全边际。

### 10.4 关键观测指标：建立新周期的监控体系

在新周期范式下，传统的能繁母猪存栏+猪粮比+仔猪价格的监控体系需要升级。建议建立以下多维度监控体系：

供给端监控：能繁母猪存栏（PSY标准化，<3750万头预警）、头部企业月度出栏量同比（<3%）、出栏均重（>130公斤二次育肥预警）。成本端监控：猪粮比（<4.5:1深度亏损）、配合饲料价格（>3400元/吨成本压力）。需求端监控：屠宰量环比变化、猪肉零售价/鸡肉零售价（>1.5替代效应）。情绪端监控：仔猪价格（15公斤<300元/头悲观信号）、生猪期货主力合约持仓量（异常增仓投机信号）。

### 10.5 前瞻：2027-2030年猪周期展望

**2027年**：若能繁母猪去化加速，下半年有望进入新一轮上行周期，全年均价13-15元/公斤。头部企业盈利修复，集中度进一步提升。

**2028-2029年**：头部企业扩张更加理性，猪价在14-18元/公斤区间波动，行业实现稳定盈利。

**2030年**：行业进入寡头竞争格局，前10家市占率超50%。猪周期波动进一步收窄（振幅<1.5倍），估值体系从PB主导转向PE主导。

### 10.6 核心结论

1. **猪周期未消失，但已变形**：2026年猪周期在PSY效率跃升、规模化主导和政策调控三大因子作用下，演变为"弱周期、窄幅波动、温和上涨/下跌"的新范式。
2. **能繁母猪存栏指标失效**：传统核心指标因PSY提升和规模化行为而预测效力大幅下降，投资者需建立包含PSY标准化产能、头部企业出栏、二次育肥比例等新指标的多维监控体系。
3. **低成本龙头是最佳投资选择**：新周期中投资胜负手不再是判断拐点 timing，而是选择具有成本优势的龙头企业。牧原股份（11.6元/公斤）、温氏股份（12.0元/公斤）、海大集团是穿越周期的最优选择。
4. **2026年下半年是左侧布局窗口**：Q3猪价磨底（9.0-10.0元/公斤），Q4温和回升至11.0-12.5元/公斤。左侧投资者可在Q3建立底仓，右侧投资者等待2027年Q1拐点确认后加仓。
5. **周期投资的预期回报需下调**：新周期中猪价波动幅度收窄、暴利期消失，养殖板块年化回报预期应从历史上的30%-50%下调至15%-25%。
6. **行业集中度提升是不可逆趋势**：到2030年前10家猪企市占率将超50%，行业从"完全竞争"走向"寡头垄断"，低成本龙头将持续受益，高成本企业将被淘汰。
