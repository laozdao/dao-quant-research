# 电子特种气体——半导体"粮食"的稀缺性与国产替代量化分析

## 摘要

电子特种气体（Electronic Specialty Gases, 简称"电子特气"）是半导体制造的"粮食"和"血液"，贯穿集成电路制造的每一个核心工艺环节。随着全球半导体产业链向中国大陆加速转移，叠加人工智能、新能源汽车、光伏等下游需求爆发，电子特气正迎来史无前例的战略性机遇。本报告以量化分析框架，系统拆解电子特气各品种的供需格局、涨价传导机制、国产替代进程，并对A股核心标的进行多因子评分与投资价值排序。

**核心结论：**

| 维度 | 关键判断 |
|------|----------|
| 行业景气度 | 2026年电子特气行业景气度评级：**A+**，处于新一轮超级上行周期起点 |
| 供需格局 | 全球需求CAGR 8.2%，国产自给率仅15%，供需缺口持续扩大 |
| 涨价弹性 | NF3涨幅40%+、WF6涨价周期启动，龙头企业毛利率弹性显著 |
| 国产替代 | 认证壁垒构筑2年+护城河，已通过认证企业享受超额收益 |
| 投资评级 | 行业整体"增持"，重点标的"买入" |

## 第一章 行业概览与战略定位

### 1.1 电子特气的定义与分类

电子特种气体是指用于半导体、光伏面板、LED等电子工业领域的特种气体，纯度要求通常达到99.999%（5N）至99.9999999%（9N）级别。根据化学成分，电子特气可分为以下几大类别：

| 类别 | 代表品种 | 主要应用 | 纯度要求 |
|------|----------|----------|----------|
| 含氟气体 | 三氟化氮(NF3)、四氟化碳(CF4)、六氟化钨(WF6) | 刻蚀、清洗、成膜 | 5N-6N |
| 含硅气体 | 硅烷(SiH4)、二氯二氢硅(SiH2Cl2) | 化学气相沉积(CVD) | 5N-6N |
| 含氮气体 | 氨气(NH3)、一氧化二氮(N2O) | 掺杂、氧化 | 5N-7N |
| 含碳气体 | 正硅酸乙酯(TEOS)、三甲基铝(TMA) | 介质膜沉积 | 5N-6N |
| 稀有混合气 | 氖氩氪氙混合气 | 光刻、离子注入 | 高精度配比 |

### 1.2 电子特气的战略定位：半导体制造的"粮食"

电子特气在半导体制造中的地位可从以下三个维度理解：

**（1）贯穿全工艺流程**

集成电路制造涉及数百个工艺步骤，其中超过30%的步骤需要使用电子特气。从晶圆制造到封装测试，电子特气无处不在：

```
晶圆制造流程中电子特气应用分布：
├── 氧化（Oxidation）─── O2、N2O
├── 化学气相沉积（CVD）── SiH4、TEOS、NH3、NF3
├── 物理气相沉积（PVD）── Ar、Kr、Xe
├── 刻蚀（Etching）───── CF4、NF3、SF6、Cl2、HBr
├── 离子注入（Implant）── BF3、AsH3、PH3
├── 掺杂（Doping）────── B2H6、PH3
├── 清洗（Cleaning）──── NF3、SF6
└── 扩散（Diffusion）─── N2、O2、Ar
```

**（2）材料成本占比高**

在集成电路制造中，电子特气占材料成本的14%左右，仅次于硅片，是第二大耗材类别。随着制程节点从28nm向7nm、5nm、3nm演进，单位晶圆的特气消耗量呈指数级增长。

| 制程节点 | 单片晶圆特气成本（美元） | 较28nm增幅 |
|----------|-------------------------|------------|
| 28nm | 约45 | 基准 |
| 14nm | 约65 | +44% |
| 7nm | 约95 | +111% |
| 5nm | 约130 | +189% |
| 3nm | 约180 | +300% |

**（3）纯度决定良率**

电子特气中哪怕含有ppb（十亿分之一）级别的杂质，都可能导致整批晶圆报废。以5nm制程为例，特气纯度每提升一个9（即从5N到6N），可将晶圆良率提升2-3个百分点。这种极致的纯度要求构筑了极高的技术壁垒。

### 1.3 市场规模与增长驱动力

#### 1.3.1 全球市场规模

2025年全球电子特气市场规模约为108亿美元，预计到2030年将突破170亿美元，复合年增长率（CAGR）为8.2%。增长的核心驱动力包括：

- **半导体产业扩张**：全球晶圆产能持续扩张，2025-2030年全球新增晶圆厂超过80座
- **先进制程升级**：3nm以下制程规模化量产，单位用量大幅提升
- **新兴应用拉动**：AI芯片（GPU/TPU）、HBM存储、车规芯片需求井喷
- **光伏面板需求**：N型光伏电池片大规模扩产拉动NF3、NH3需求

#### 1.3.2 中国市场规模

中国是全球最大的电子特气消费市场，2025年国内市场规模约320亿元人民币，占全球比重约42%。但国产自给率仅约15%，意味着每年进口额超过270亿元。国产替代空间巨大。

|------|---------------------|----------|------------|
| 2020 | 180 | 35% | 8% |
| 2022 | 240 | 38% | 11% |
| 2024 | 290 | 40% | 13% |

### 2.2 六氟化钨（WF6）：薄膜沉积的贵族气体

#### 2.2.1 基本属性

六氟化钨（WF6）是钨的含氟化合物，常温下为无色气体，遇潮气分解。主要用于半导体制造中的化学气相沉积（CVD）和原子层沉积（ALD），用于沉积钨金属互连层。

- 预计2030年全球需求将达2,500吨，CAGR约13%

### 2.3 硅烷（SiH4）：CVD沉积的核心原料

#### 2.3.1 基本属性与供需

硅烷是半导体、光伏、面板三大领域通用的核心气体，主要用于CVD薄膜沉积。2025年全球硅烷需求量约4.5万吨，中国占比约45%。

### 2.4 氨气（NH3）：掺杂与氧化的关键介质

高纯氨气（5N-7N级别）在半导体制造中用于氮化硅薄膜沉积（Si3N4）和P型掺杂。2025年全球半导体级氨气需求量约8万吨，中国占比约35%。

### 2.5 正硅酸乙酯（TEOS）：先进封装的增量品种

TEOS主要用于SiO2薄膜的CVD沉积，随着先进封装（2.5D/3D封装、Chiplet）需求爆发，TEOS需求增速显著提升至12%以上。

### 4.3 壁垒体系分析

电子特气行业的壁垒可概括为"四高"：

**（1）技术壁垒极高**

- 纯化技术：将工业级气体提纯至5N-9N级别，涉及数十道精密纯化工艺
- 分析检测：杂质检测精度要求达到ppt（万亿分之一）级别
- 包装储运：特殊阀门、内表面处理、防二次污染技术
- 配气技术：多组分混合气的高精度配比（误差<0.5%）

**（2）认证壁垒极高**

- 客户认证周期：1-2年（晶圆代工厂）
- 认证流程：小试→中试→量产→定期复审
- 认证成本：数百万元
- 认证通过率：<30%

**（3）资金壁垒高**

- 新建一条电子特气生产线投资通常在2-5亿元
- 配套检测设备投资数千万元
- 产能爬坡期通常2-3年

**（4）客户粘性极高**

- 晶圆厂一旦完成供应商认证，极少更换（工艺兼容性风险）
- 替换成本极高（整线验证需6-12个月）
- 长协合同通常3-5年，价格粘性强

**壁垒量化评分表：**

| 壁垒维度 | 评分（1-10） | 对比行业 |
|----------|-------------|----------|
| 技术壁垒 | 9.5 | 高于锂电池材料(8) |
| 认证壁垒 | 9.8 | 高于航空零部件(9) |
| 资金壁垒 | 7.5 | 高于普通化工(5) |
| 客户粘性 | 9.0 | 高于半导体设备(8.5) |
| **综合壁垒评分** | **9.0** | **极高** |

### 5.2 国产替代路径图


## 第六章 价格周期与盈利弹性分析

### 6.1 电子特气价格周期特征

电子特气价格具有典型的"成本驱动+需求弹性"双因子特征：

**价格形成机制：**

```
电子特气价格 = 基础成本 × (1 + 加工费率) × (1 + 品牌溢价) × 供需因子

其中：
- 基础成本：原料成本，占60-70%
- 加工费率：提纯/配比/包装，占25-35%
- 品牌溢价：国际品牌溢价20-30%
- 供需因子：供需紧张时1.1-1.5x
```
