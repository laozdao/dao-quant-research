# A股情绪量化方法论深度研究：大盘·行业·概念·个股四层体系

> **一句话摘要**：情绪是A股市场的"隐形之手"——构建从大盘到个股的四层情绪量化体系，将主观感受转化为可计算、可回测、可交易的客观信号。

---

## 一、引言/背景

### 1.1 为什么情绪量化在A股特别重要？

A股市场具有两个鲜明特征：**政策市**和**散户市**。

- **政策市**：政策变化直接驱动板块轮动，情绪对政策信号的反应速度远超基本面
- **散户市**：散户交易占比高，追涨杀跌行为放大市场波动，情绪偏差创造超额收益机会

> 传统炒股靠打听消息、看K线图，情绪量化靠数据说话。将"感觉市场很热"转化为"情绪指数处于高潮区间（>80分）"，才能做出纪律化决策。

### 1.2 研究目标

本文旨在：
1. 构建覆盖**大盘、行业、概念、个股**四层的情绪量化体系
2. 系统梳理100+情绪指标的分类与计算方法
3. 建立情绪周期模型（冰点→修复→高潮→分歧→再冰点）
4. 提供可落地的Python实现框架与实战应用场景

## 三、第一层：大盘情绪量化

### 3.1 大盘情绪指标矩阵

#### 量价类指标

| 指标 | 计算方法 | 情绪含义 | 极端阈值 |
|------|----------|----------|:--------:|
| **涨跌家数比** | 上涨家数 / 下跌家数 | 市场广度 | >3 或 <0.33 |
| **涨跌停比** | 涨停家数 / 跌停家数 | 极端情绪强度 | >5 或 <0.2 |
| **涨停家数** | 当日涨停股票总数 | 市场活跃度 | >80（高潮）<20（冰点） |
| **炸板率** | 炸板数 / (涨停数+炸板数) | 情绪持续性 | >40%（分歧）<15%（一致） |
| **成交额** | 全市场成交金额 | 资金参与度 | >1.5万亿（过热）<5000亿（冰点） |
| **换手率** | 全市场加权平均换手率 | 交易活跃度 | >5%（过热）<1%（冷清） |

#### 资金类指标

| 指标 | 计算方法 | 情绪含义 | 极端阈值 |
|------|----------|----------|:--------:|
| **北向资金净流入** | 沪股通+深股通净买入额 | 外资情绪方向 | 单日>200亿（极度乐观） |
| **融资余额变化** | 当日融资余额 - 前日 | 杠杆情绪 | 连续5日增加（加杠杆） |
| **融资买入额占比** | 融资买入额 / 成交额 | 杠杆参与度 | >10%（激进）<6%（保守） |
| **ETF申赎情况** | 股票ETF净申购/赎回份额 | 散户逆向情绪 | 大额净申购（底部信号） |
| **主力资金净流入** | 大单净买入额 | 机构方向 | — |

#### 衍生类指标

| 指标 | 计算方法 | 情绪含义 |
|------|----------|----------|
| **创近一年新高/新低占比** | 创新高家数 / 总家数 | 市场广度与趋势强度 |
| **基金股票仓位** | 开放式基金估算股票投资比例 | 机构情绪 | 
| **股债性价比（风险溢价）** | (1/PE - 无风险利率) | 资产比价 |
| **新增开户数** | 新增投资者账户数 | 散户入场意愿 |
| **偏股基金发行规模** | 新成立偏股基金份额 | 散户资金供给 |

### 3.2 大盘恐慌贪婪指数构建

#### 综合评分方法

采用**投票法**（等权投票）构建大盘情绪综合指数：

```
大盘情绪指数 = (量价得分 + 资金得分 + 衍生得分) / 3

每个子维度：
  信号看多 → +1分
  信号中性 → 0分
  信号看空 → -1分

最终映射到 0-100 分：
  0-20：极度恐慌（冰点）
  20-40：偏恐慌
  40-60：中性
  60-80：偏贪婪
  80-100：极度贪婪（高潮）
```

#### 情绪周期模型

```
     高潮(80-100)
        ↑
  修复(40-60) → 分歧(60-80)
        ↑              ↓
  冰点(0-20) ←────────┘
```

| 阶段 | 情绪指数 | 市场特征 | 操作建议 |
|------|:--------:|----------|----------|
| **冰点** | 0-20 | 涨停<20家，成交额<5000亿，融资余额下降 | **逆向布局**，分批建仓 |
| **修复** | 20-40 | 涨停回升，成交额温和放大，北向资金转净流入 | **持有或加仓**，跟随主线 |
| **高潮** | 60-80 | 涨停>80家，成交额>1.5万亿，融资余额快速上升 | **逐步减仓**，锁定利润 |
| **分歧** | 40-60（从高位回落） | 炸板率>40%，涨跌停比下降，主力资金流出 | **清仓或对冲**，规避回调 |

## 四、第二层：行业情绪量化

### 4.1 行业情绪指标

| 指标 | 计算方法 | 情绪含义 |
|------|----------|----------|
| **行业涨跌幅** | 行业指数日涨跌幅 | 当日情绪方向 |
| **行业成交额占比** | 行业成交额 / 全市场成交额 | 资金聚焦度 |
| **行业换手率** | 行业内个股加权平均换手率 | 交易活跃度 |
| **行业涨跌家数比** | 行业内上涨家数 / 下跌家数 | 行业广度 |
| **行业北向资金持仓变化** | 行业北向持股变化量 | 外资行业偏好 |
| **行业融资余额变化** | 行业内个股融资余额合计变化 | 杠杆行业偏好 |
| **行业相对强弱（RS）** | 行业指数 / 大盘指数（N日） | 相对表现 |

### 4.2 行业情绪轮动模型

#### 资金偏离度

```
行业资金偏离度 = (行业成交额占比 - 行业市值占比) / 行业市值占比

偏离度 > 0：资金超配（情绪过热）
偏离度 < 0：资金低配（情绪冷清）
```

#### 行业情绪得分

```
行业情绪得分 = 0.3 × 涨跌家数比得分
             + 0.2 × 换手率偏离得分
             + 0.2 × 北向资金变化得分
             + 0.15 × 融资余额变化得分
             + 0.15 × 相对强弱得分
```

### 4.3 行业轮动信号

| 信号 | 条件 | 含义 |
|------|------|------|
| **行业启动** | 行业情绪得分连续3日上升 + 成交额占比突破阈值 | 资金开始流入 |
| **行业加速** | 行业RS创新高 + 换手率持续放大 | 主升浪 |
| **行业过热** | 行业资金偏离度 > 2倍标准差 | 短期见顶风险 |
| **行业退潮** | 行业情绪得分连续3日下降 + 成交额占比萎缩 | 资金撤离 |

## 五、第三层：概念情绪量化

### 5.1 概念情绪指标

概念板块（主题投资）是A股最具情绪化特征的交易维度。

| 指标 | 计算方法 | 情绪含义 |
|------|----------|----------|
| **概念涨跌幅** | 概念指数日涨跌幅 | 当日情绪方向 |
| **概念涨停家数** | 概念内涨停个股数量 | 情绪强度 |
| **概念成交额** | 概念内个股成交额合计 | 资金参与度 |
| **概念换手率** | 概念内个股加权换手率 | 交易活跃度 |
| **龙头股连板数** | 概念内最高连板天数 | 情绪高度 |
| **概念持续性** | 概念连续上涨天数 | 情绪持久度 |
| **概念龙头封板强度** | 龙头股封单金额 / 成交额 | 情绪确定性 |

### 5.2 概念热度生命周期模型

```
萌芽期 → 发酵期 → 高潮期 → 分歧期 → 退潮期
  │          │          │          │          │
  │    龙头首板   涨停>5家   涨停>15家  炸板率>40%  涨停<3家
  │    成交温和   资金流入    成交爆发    龙头分歧    资金流出
  │    无媒体报道  开始传播    全网刷屏    负面出现    媒体遗忘
  │          │          │          │          │
  ▼          ▼          ▼          ▼          ▼
潜伏观望    适度参与    谨慎追高    开始减仓    完全回避
```

#### 各阶段量化特征

| 阶段 | 涨停家数 | 成交额趋势 | 龙头表现 | 媒体热度 | 操作 |
|------|:--------:|:----------:|:--------:|:--------:|------|
| **萌芽期** | 1-3 | 温和 | 首板或2板 | 低 | **观察，小仓试错** |
| **发酵期** | 3-8 | 放大 | 3板+ | 上升 | **加仓，跟随龙头** |
| **高潮期** | 8-20 | 剧增 | 5板+ | 爆发 | **持有，不追高** |
| **分歧期** | 下降 | 萎缩 | 炸板/断板 | 顶峰 | **减仓，锁定利润** |
| **退潮期** | <3 | 低迷 | 补跌 | 消退 | **回避，等待新周期** |

### 5.3 概念情绪综合评分

```
概念情绪得分 = 0.25 × 涨停强度得分
             + 0.20 × 资金参与度得分
             + 0.20 × 龙头连板得分
             + 0.15 × 持续性得分
             + 0.10 × 封板强度得分
             + 0.10 × 媒体热度得分
```

## 六、第四层：个股情绪量化

### 6.1 个股情绪指标

| 指标 | 计算方法 | 情绪含义 |
|------|----------|----------|
| **连板天数** | 连续涨停天数 | 情绪高度（短线核心指标） |
| **封板强度** | 封单金额 / 成交额 | 多头决心 |
| **炸板次数** | 当日打开涨停次数 | 多空博弈激烈度 |
| **换手率** | 当日换手率 | 交易活跃度 |
| **量比** | 当日成交量 / 5日均量 | 资金关注度 |
| **龙虎榜买入额** | 龙虎榜机构/游资买入金额 | 主力动向 |
| **北向资金持股变化** | 个股北向持股变化量 | 外资态度 |
| **融资余额变化率** | (今日融资余额 - 昨日) / 昨日 | 杠杆情绪 |
| **大单净买入占比** | 大单净买入 / 成交额 | 机构方向 |
| **席位溢价** | 龙虎榜知名游资/机构买入 | 情绪溢价 |

### 6.2 个股情绪评分模型

```
个股情绪得分 = 0.20 × 连板高度得分
             + 0.15 × 封板强度得分
             + 0.15 × 量比得分
             + 0.15 × 换手率得分
             + 0.10 × 龙虎榜得分
             + 0.10 × 资金流向得分
             + 0.10 × 北向/融资得分
             + 0.05 × 席位溢价得分
```

### 6.3 个股情绪分类

| 情绪类型 | 特征 | 操作建议 |
|----------|------|----------|
| **龙头股** | 连板最高、封板最强、市场辨识度高 | 优先关注，回调低吸 |
| **补涨股** | 同概念内尚未启动、基本面有支撑 | 萌芽期布局 |
| **跟风股** | 跟随龙头上涨但无独立逻辑 | 回避，龙头分歧时最先补跌 |
| **反包股** | 前日炸板/跌停后反包涨停 | 高风险高回报，轻仓博弈 |
| **庄股** | 换手率极低、走势独立 | 回避，流动性风险极高 |

## 七、四层联动：情绪传导机制

### 7.1 自上而下的情绪传导

```
大盘情绪（环境）
    │
    ├── 决定整体仓位水平
    │   冰点 → 加仓至80%
    │   高潮 → 降仓至30%
    │
    ▼
行业情绪（方向）
    │
    ├── 决定行业配置权重
    │   启动期 → 标配
    │   加速期 → 超配
    │   过热期 → 低配
    │
    ▼
概念情绪（主题）
    │
    ├── 决定个股选择范围
    │   发酵期 → 买入龙头
    │   高潮期 → 持有不追
    │   退潮期 → 全部卖出
    │
    ▼
个股情绪（执行）
    │
    ├── 决定买卖时点与仓位
    │   高情绪得分 → 大仓位
    │   低情绪得分 → 小仓位或不参与
```

### 7.2 情绪背离信号

情绪背离是最有价值的反向信号：

| 背离类型 | 定义 | 含义 | 操作 |
|----------|------|------|------|
| **量价背离** | 指数创新高但成交额下降 | 上涨动力不足 | 减仓 |
| **广度背离** | 指数涨但涨跌家数比下降 | 少数权重股拉动 | 警惕 |
| **资金背离** | 指数涨但主力资金净流出 | 机构在出货 | 减仓 |
| **行业背离** | 大盘涨但行业情绪得分下降 | 轮动信号 | 调仓 |
| **个股背离** | 个股涨停但封板强度下降 | 多头力竭 | 次日卖出 |

## 八、Python实现框架

### 8.1 数据源

| 数据类型 | 推荐数据源 | Python库 |
|----------|------------|----------|
| 行情数据 | 东方财富、同花顺 | `akshare`, `qstock`, `tushare` |

### 8.2 核心代码框架

```python
import akshare as ak
import pandas as pd
import numpy as np

class MarketSentiment:
    """A股四层情绪量化系统"""
    
    def __init__(self):
        self.data = {}
    
    # ========== 第一层：大盘情绪 ==========
    def get_market_sentiment(self, date):
        """计算大盘情绪综合得分（0-100）"""
        # 量价指标
        limit_up, limit_down = self._get_limit_data(date)
        adv, dec = self._get_adv_dec_data(date)
        total_amount = self._get_total_amount(date)
        
        # 资金指标
        north_flow = self._get_north_flow(date)
        margin_change = self._get_margin_change(date)
        
        # 衍生指标
        new_high_ratio = self._get_new_high_ratio(date)
        
        # 综合评分
        score = self._calc_composite_score(
            limit_up, limit_down, adv, dec,
            total_amount, north_flow, margin_change, new_high_ratio
        )
        return score
    
    # ========== 第二层：行业情绪 ==========
    def get_industry_sentiment(self, date):
        """计算行业情绪得分排名"""
        industries = self._get_all_industries()
        scores = {}
        for ind in industries:
            scores[ind] = self._calc_industry_score(ind, date)
        return sorted(scores.items(), key=lambda x: x[1], reverse=True)
    
    # ========== 第三层：概念情绪 ==========
    def get_concept_sentiment(self, date):
        """计算概念情绪得分及生命周期阶段"""
        concepts = self._get_all_concepts()
        results = {}
        for concept in concepts:
            score = self._calc_concept_score(concept, date)
            stage = self._identify_concept_stage(concept, date)
            results[concept] = {'score': score, 'stage': stage}
        return results
    
    # ========== 第四层：个股情绪 ==========
    def get_stock_sentiment(self, stock_code, date):
        """计算个股情绪综合得分"""
        return self._calc_stock_score(stock_code, date)
    
    # ========== 情绪周期判断 ==========
    def get_market_cycle(self, date):
        """判断当前市场情绪周期阶段"""
        score = self.get_market_sentiment(date)
        if score < 20:
            return '冰点'
        elif score < 40:
            return '修复'
        elif score < 60:
            return '中性'
        elif score < 80:
            return '高潮'
        else:
            return '极度贪婪'
```

### 8.3 回测验证要点

| 回测维度 | 验证方法 | 预期目标 |
|----------|----------|----------|
| 大盘择时 | 情绪冰点买入、高潮卖出 | 年化收益跑赢基准 |
| 行业轮动 | 超配高情绪行业、低配低情绪行业 | 超额收益稳定 |
| 概念捕捉 | 发酵期买入、退潮期卖出 | 胜率>60% |
| 个股筛选 | 高情绪得分个股组合 | 相对收益为正 |

## 九、实战应用场景

### 9.1 场景一：大盘择时

| 大盘情绪 | 仓位建议 | 配置方向 |
|----------|:--------:|----------|
| 冰点（0-20） | **70-80%** | 逆指数布局，关注超跌板块 |
| 修复（20-40） | **50-70%** | 顺指数布局，跟随主线 |
| 中性（40-60） | **40-50%** | 均衡配置，精选个股 |
| 高潮（60-80） | **20-40%** | 逐步减仓，锁定利润 |
| 极度贪婪（80-100） | **0-20%** | 清仓或对冲，规避回调 |

### 9.2 场景二：行业轮动

- **行业启动信号**：行业情绪得分连续3日上升 + 成交额占比突破近20日均值
- **行业加速信号**：行业RS创20日新高 + 换手率持续3日放大
- **行业过热信号**：行业资金偏离度 > 2倍标准差 + 涨跌家数比开始下降

### 9.3 场景三：概念炒作节奏

- **萌芽期**：龙头首板出现，小仓位试错（总仓位5%）
- **发酵期**：涨停家数增至5+，加仓至10-15%
- **高潮期**：全网刷屏，持有不追，开始设止盈
- **分歧期**：炸板率飙升，减仓至5%以下
- **退潮期**：完全回避，等待下一个概念周期

### 9.4 场景四：个股情绪交易

- **龙头战法**：概念发酵期买入最高连板龙头，分歧期卖出
- **补涨战法**：概念高潮期寻找同概念内尚未启动的低位个股
- **反包战法**：前日炸板个股次日反包涨停，轻仓博弈（仓位<3%）

## 十、风险提示与局限性

### 10.1 情绪量化的局限性

1. **历史不代表未来**：情绪阈值基于历史统计，市场结构变化可能导致失效
2. **政策突变风险**：突发政策可能瞬间改变情绪方向，模型无法预判
3. **数据滞后性**：部分指标（如融资余额）存在T+1滞后
4. **过度拟合风险**：参数优化过度可能导致回测表现优异但实战失效
5. **小样本问题**：极端行情样本有限，统计意义可能不足

### 10.2 使用原则

1. **情绪是辅助工具，不是唯一依据**：必须结合基本面、技术面综合判断
2. **逆向操作需要勇气和纪律**：冰点买入需要承受短期浮亏
3. **仓位管理比择时更重要**：情绪信号决定方向，仓位控制决定生存
4. **定期更新参数**：每季度回顾情绪阈值的有效性，动态调整

## 十一、结论

### 11.1 核心结论

1. **A股情绪可量化**：通过100+指标的五大数据维度，可以将主观的"市场感觉"转化为客观的情绪得分
2. **四层体系缺一不可**：大盘定仓位、行业定方向、概念定主题、个股定标的，层层递进
3. **情绪周期可识别**："冰点→修复→高潮→分歧→再冰点"的周期模型提供了可操作的交易节奏
4. **情绪背离最有价值**：量价背离、广度背离、资金背离是最佳的反向信号
5. **情绪是辅助而非全部**：必须与基本面分析、风险控制体系结合使用

### 11.2 方法论价值

> 情绪量化的核心价值不在于预测涨跌，而在于**将主观感受客观化、将模糊判断精确化、将交易决策纪律化**。
