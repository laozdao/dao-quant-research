# A股各板块轮动规律的量化分析方法

## 摘要

板块轮动是A股市场长期存在的核心现象，不同行业、风格、市值板块在不同经济周期和市场环境下交替表现强势。本文系统梳理A股板块轮动的四大维度（行业轮动、风格轮动、大小盘轮动、主题轮动），从经济周期驱动框架出发，深入解析六大量化轮动模型（动量轮动、均值回归、相对强弱RSI、均线能量、宏观因子预测、多策略融合），并提供ETF轮动实战策略、回测框架及Python实现代码，最终形成一套可落地的板块轮动量化投资工具箱。

## 第一章 板块轮动的核心概念与A股特征

### 1.1 什么是板块轮动

板块轮动（Sector Rotation）是指不同行业或风格板块在时间上交替表现强势与弱势的现象。其本质是市场资金在不同板块间流动，追逐当前最具吸引力的投资机会。

**核心逻辑**：不同行业的盈利周期、对宏观经济的敏感度、估值水平存在差异，导致其在不同市场环境下表现分化。

### 1.2 A股板块轮动的四大维度

| 轮动维度 | 划分标准 | 典型板块/风格 | 轮动周期 |
|---------|---------|-------------|---------|
| 行业轮动 | 申万一级行业 | 电子、医药、银行、煤炭等31个行业 | 1-6个月 |
| 风格轮动 | 成长/价值 | 成长股（科技/新能源）vs 价值股（银行/公用事业） | 3-12个月 |
| 大小盘轮动 | 市值规模 | 上证50（大盘）vs 中证1000（小盘） | 6-24个月 |
| 主题轮动 | 概念/政策 | AI、机器人、低空经济、跨境支付等 | 1-3个月 |

### 1.3 A股板块轮动的核心特征

**特征一：跷跷板效应显著**

20年来A股市场存在很强的跷跷板效应[1]：
- 上证50表现好 → 中小创表现差
- 非周期消费白马走强 → 周期股走弱
- 金融板块强势 → 科技板块承压

**特征二：四种主体风格轮换**

A股长久以来呈现四种主体风格的轮换[1]：

| 风格 | 代表板块 | 发力时点 | 历史案例 |
|------|---------|---------|---------|
| 蓝筹 | 上证50、国字头 | 牛市初期（拉指数）和末期（护盘） | 2013年1849-3406 |
| 概念 | 中小创、主题 | 穿插全程，波动最大 | 2015年3406-5178 |
| 周期 | 有色、券商、煤炭 | 经济扩张期 | 2005-2007年998-6124 |
| 非周期 | 消费白马、医药 | 牛市末期补涨+熊市上半场防御 | 2017-2021年白马行情 |

**特征三：资金驱动的六道轮回**

板块轮动的深层驱动力是资金的循环规律[1]：

```
资金循环六阶段
1. 泉眼无声惜细流 → 资金慢慢流入，关注度低（最佳买入点）
2. 桃花流水鳜鱼肥 → 资金汇聚成池，岁月静好
3. 万山红遍百舸争流 → 大水漫灌，钱多人傻（最佳卖出点）
4. 飞流直下三千尺 → 资金出逃，财富蒸发
5. 花自飘零水自流 → 资金持续流出
6. 物是人非事事休 → 资金枯竭，繁华落尽（逆向买入点）
```

## 第二章 经济周期驱动框架

### 2.1 美林投资时钟与A股行业轮动

美林投资时钟将经济周期分为四个阶段，每个阶段对应不同的优势行业[2]：

| 经济阶段 | 经济增长 | 通胀 | 优势板块 | A股历史表现 |
|---------|---------|------|---------|------------|
| 复苏 | ↑ | ↓ | 科技、券商、可选消费 | 2009年、2019年初 |
| 过热 | ↑ | ↑ | 周期（有色、煤炭、钢铁） | 2007年上半年、2021年 |
| 滞胀 | ↓ | ↑ | 公用事业、医药、必需消费 | 2011年、2018年 |
| 衰退 | ↓ | ↓ | 债券、银行、高股息 | 2008年、2015年股灾 |

**A股本土化修正**：
- 2017年后宏观经济弹性和波动下降，行业分化加剧
- 行业周期的独立性强于宏观周期的同步性
- 政策驱动（如新能源补贴、AI政策）成为独立于经济周期的轮动力量

### 2.2 库存周期与行业轮动

库存周期（基钦周期，3-4年）对行业轮动有重要指导意义：

| 库存阶段 | 特征 | 优势行业 | 劣势行业 |
|---------|------|---------|---------|
| 被动去库存 | 需求回升，库存被动下降 | 科技、可选消费 | 公用事业 |
| 主动补库存 | 企业扩产，量价齐升 | 周期、制造 | 防御 |
| 被动补库存 | 需求下降，库存被动积压 | 公用事业、医药 | 周期 |
| 主动去库存 | 企业减产，去库为主 | 银行、高股息 | 制造 |

### 2.3 六大板块景气度模型

金融界研究院将A股划分为六大板块，通过景气度指数研判轮动规律[3]：

| 板块 | 涵盖行业 | 景气度特征 |
|------|---------|-----------|
| 金融 | 银行、证券、保险 | 牛市初期率先启动，末期护盘 |
| 科技 | 新能源、半导体、信息技术 | 政策驱动，反弹持续性最强 |
| 制造 | 工程机械、化工、造纸 | 跟随经济周期，弹性中等 |
| 防御 | 食品饮料、酿酒、医药 | 熊市抗跌，牛市补涨 |
| 消费 | 家电、旅游、农林牧渔 | 年末/政策刺激时发力 |
| 顺周期 | 煤炭、钢铁、有色 | PPI高位时表现最佳 |

**2022年轮动案例**[3]：
- 2-3月：顺周期板块景气度100%，煤炭"煤飞色舞"
- 5-8月：科技板块长尾效应，景气度维持80%+
- 12月：消费板块接力，政策释放扩大内需信号

## 第三章 六大量化轮动模型

### 3.1 模型一：动量轮动模型

**核心逻辑**：过去强势的行业在未来一段时间继续保持强势（动量效应）。

**构建方法**：

```python
def momentum_score(prices, lookback=20):
    """
    计算行业动量得分
    prices: 行业指数价格序列
    lookback: 回望期（交易日）
    """
    returns = prices.iloc[-1] / prices.iloc[-lookback] - 1
    return returns.rank(pct=True)
```

**策略规则**：
- 每月/每周计算各行业过去N日收益率
- 买入动量排名Top K的行业
- 等权或按动量加权配置

**优缺点**：

| 优点 | 缺点 |
|------|------|
| 逻辑简单，易于实现 | 震荡市频繁切换，损耗大 |
| 趋势市中表现优异 | 动量崩溃时回撤大 |
| 参数少，过拟合风险低 | 对参数敏感 |

### 3.2 模型二：均值回归轮动模型

**核心逻辑**：过度上涨的行业会回调，过度下跌的行业会反弹（反转效应）。

**构建方法**：

```python
def mean_reversion_score(prices, lookback=20):
    """
    计算行业均值回归得分
    得分越高表示超涨越严重（应卖出），得分越低表示超跌越严重（应买入）
    """
    ma = prices.rolling(lookback).mean()
    deviation = (prices.iloc[-1] - ma.iloc[-1]) / ma.iloc[-1]
    return deviation.rank(pct=True)
```

**策略规则**：
- 计算各行业价格相对移动平均线的偏离度
- 买入偏离度最低（超跌）的行业
- 卖出偏离度最高（超涨）的行业

**适用场景**：震荡市、行业分化剧烈时

### 3.3 模型三：相对强弱RSI轮动模型

**核心逻辑**：通过相对强弱指标（RSI）识别行业的超买超卖状态。

**RSI计算**：

$$RSI = 100 - \frac{100}{1 + RS}$$

其中 $RS = \frac{n日内上涨日平均涨幅}{n日内下跌日平均跌幅}$

**策略规则**[4]：

| RSI区间 | 信号 | 操作 |
|--------|------|------|
| RSI > 90 | 极度强势 | 持有/谨慎追高 |
| 70 < RSI < 90 | 强势 | 持有 |
| 50 < RSI < 70 | 中性偏强 | 关注 |
| 30 < RSI < 50 | 中性偏弱 | 观望 |
| RSI < 30 | 弱势 | 回避/等待反转 |

**2024年RS>90的行业**[4]：煤炭、电力及公用事业、家电、银行、石油石化、通信、有色金属、农林牧渔、汽车

### 3.4 模型四：均线能量轮动模型

**核心逻辑**：通过均线系统的"能量"判断行业趋势的强度和持续性。

**行业轮动A策略**[5]：

| 参数 | 设定 |
|------|------|
| 标的池 | 证券公司、半导体、CS新能车、家用电器、中证医疗、中证红利 |
| 轮动指标 | 均线能量（20日均线） |
| 过滤条件 | 均线能量 > 0 |
| 缓冲机制 | 排名掉落到第三名后才卖出 |
| 持仓数量 | 排名前两位 |

**回测表现（2013-2026）**[5]：

| 指标 | 数值 |
|------|------|
| 总收益 | 1859.6% |
| 年化收益率 | 24.9% |
| 最大回撤 | 37.6% |
| 收益回撤比 | 0.66 |

**关键改进**：
- **过滤**：过滤掉熊市，尽量不在熊市买入
- **缓冲**：排名变化足够大时才交易，减少频繁切换

### 3.5 模型五：宏观因子预测模型

**核心逻辑**：通过宏观经济指标预测大小盘/成长价值的风格走向。

**大小盘风格轮动因子**[6]：

| 因子 | 含义 | 对风格的影响 |
|------|------|------------|
| M2同比增速 | 货币流动性 | M2↑ → 小盘占优 |
| PPI同比增速 | 生产环节通胀 | PPI↑ → 周期/大盘占优 |
| 大小盘波动率之比 | 市场情绪 | 比值↑ → 大盘情绪强 |

**预测模型**：

$$D(R_t) = \alpha + \beta_1 \cdot MG_{t-1} + \beta_2 \cdot PG_{t-3} + \beta_3 \cdot \sigma_{t-3} + \epsilon_t$$

其中 $D(R_t)$ 为当月小/大盘收益率差。

**实证结果**[6]：
- 78个月预测期，准确率约53.85%
- 轮动策略累计收益率307.16%，同期上证综指81.26%
- 2007年初至2010年底，累计收益率458.65%

### 3.6 模型六：多策略融合轮动模型

**核心逻辑**：单一策略难以适应所有市场环境，通过多策略融合提升稳健性。

**中信证券多策略轮动框架**[7]：

**Step 1: 策略库构建**

| 策略类型 | 代表策略 | 风格倾向 |
|---------|---------|---------|
| 预期风险 | 业绩超预期、领先上调 | 大盘均衡 |
| 预期边际改善 | 文本事件、稳健 | 大盘价值 |
| 高质量经营 | 高质量经营、红利低波 | 大盘价值 |
| 科技成长 | 科技研发 | 小盘成长 |
| 价值洼地 | 价值洼地、GARP | 小盘价值 |

**Step 2: 风格矩阵定位**

以总市值和PB将策略定位到3×3风格矩阵：

```
        价值      均衡      成长
大盘   红利低波   高质量    科技研发
中盘   PB-ROE    自由现金流  GARP
小盘   价值洼地   进取策略   科技研发
```

**Step 3: 风格动量轮动**

- 基于风格动量选择排名前三的风格
- 在风格信号中占比最高的三个策略构建组合

**回测表现（2016-2025）**[7]：

|------|------|

### 4.2 单ETF动量轮动策略

**策略逻辑**[8]：采用年化收益 × R²（拟合优度）作为动量得分，始终持有得分最高的1只ETF。

**ETF标的池示例**：

| ETF | 代码 | 代表风格 |
|-----|------|---------|
| 黄金ETF | 518880.SH | 大宗商品 |
| 纳指100 | 513100.SH | 海外资产 |
| 创业板100 | 159915.SZ | 成长股/科技 |
| 上证180 | 510180.SH | 价值股/蓝筹 |
| 中证500 | 510500.SH | 中小盘 |

**动量得分计算**：

```python
def calculate_momentum_score(prices, lookback=25):
    """
    动量得分 = 年化收益率 × R²
    """
    y = np.log(prices)  # 对数价格
    x = np.arange(len(y))
    slope, intercept = np.polyfit(x, y, 1)
    
    # 年化收益率
    annualized_returns = (np.exp(slope) ** 250) - 1
    
    # R²拟合优度
    y_pred = slope * x + intercept
    ss_res = np.sum((y - y_pred) ** 2)
    ss_tot = np.sum((y - np.mean(y)) ** 2)
    r_squared = 1 - ss_res / ss_tot
    
    return annualized_returns * r_squared
```

**策略特点**：
- R²剔除趋势不稳定的噪音
- 始终满仓，不择时
- 适合趋势明确的市场

### 5.2 轮动模型实现

```python
class SectorRotationModel:
    """板块轮动量化模型"""
    
    def __init__(self, prices_df, lookback=20):
        self.prices = prices_df
        self.lookback = lookback
        self.returns = prices_df.pct_change()
    
    def momentum_score(self):
        """动量得分：过去N日收益率排名"""
        momentum = self.prices.iloc[-1] / self.prices.iloc[-self.lookback] - 1
        return momentum.rank(pct=True)
    
    def mean_reversion_score(self):
        """均值回归得分：相对均线偏离度"""
        ma = self.prices.rolling(self.lookback).mean()
        deviation = (self.prices.iloc[-1] - ma.iloc[-1]) / ma.iloc[-1]
        return deviation.rank(pct=True)
    
    def rsi_score(self, period=14):
        """RSI得分"""
        delta = self.prices.diff()
        gain = delta.where(delta > 0, 0).rolling(period).mean()
        loss = (-delta.where(delta < 0, 0)).rolling(period).mean()
        rs = gain / loss
        rsi = 100 - (100 / (1 + rs))
        return rsi.iloc[-1]
    
    def ma_energy_score(self, ma_period=20):
        """均线能量得分"""
        ma = self.prices.rolling(ma_period).mean()
        energy = (self.prices.iloc[-1] - ma.iloc[-1]) / ma.iloc[-1]
        return energy
    
    def composite_score(self, weights=None):
        """综合得分"""
        if weights is None:
            weights = {'momentum': 0.4, 'mean_reversion': 0.2, 
                      'rsi': 0.2, 'ma_energy': 0.2}
        
        scores = pd.DataFrame({
            'momentum': self.momentum_score(),
            'mean_reversion': self.mean_reversion_score(),
            'rsi': self.rsi_score(),
            'ma_energy': self.ma_energy_score()
        })
        
        composite = sum(scores[col] * weight 
                       for col, weight in weights.items())
        return composite.rank(pct=True)
```

### 5.3 回测框架

```python
def backtest_rotation(prices_df, model, top_n=2, rebalance_freq='W'):
    """
    板块轮动回测框架
    prices_df: 行业指数价格数据
    model: 轮动模型实例
    top_n: 持仓数量
    rebalance_freq: 调仓频率（W=周，M=月）
    """
    portfolio_value = [1.0]
    current_holdings = []
    
    for date in prices_df.resample(rebalance_freq).last().index:
        # 计算得分
        scores = model.composite_score()
        top_sectors = scores.nlargest(top_n).index.tolist()
        
        # 调仓
        if set(top_sectors) != set(current_holdings):
            # 卖出非目标持仓
            # 买入目标持仓（等权）
            current_holdings = top_sectors
        
        # 计算收益
        period_return = prices_df.loc[date].pct_mean()
        portfolio_value.append(portfolio_value[-1] * (1 + period_return))
    
    return pd.Series(portfolio_value)
```

## 第六章 策略优化与风控

### 6.1 过滤机制

**问题**：熊市中即便是最强行业也会下跌。

**解决方案**：

| 过滤条件 | 触发条件 | 操作 |
|---------|---------|------|
| 大盘趋势过滤 | 沪深300 < 20日均线 | 空仓或降低仓位 |
| 波动率过滤 | VIX > 历史80%分位 | 空仓 |
| 行业得分过滤 | 所有行业得分 < 0 | 空仓 |
| 拥挤度过滤 | 行业成交额占比 > 历史95%分位 | 回避 |

### 6.2 缓冲机制

**问题**：行业排名频繁变动导致反复交易。

**解决方案**：

| 缓冲类型 | 说明 | 效果 |
|---------|------|------|
| 排名缓冲 | 排名掉出Top 3才卖出 | 减少50%+切换次数 |
| 得分缓冲 | 得分变化 > 阈值才调仓 | 降低噪音交易 |
| 时间缓冲 | 最少持有N天 | 避免过度交易 |

### 6.3 拥挤度监控

**高拥挤行业特征**[10]：
- 成交额占比处于历史高位
- 换手率异常放大
- 融资买入占比过高
- 分析师覆盖度突然增加

**2026年6月高拥挤行业**[10]：电子行业处于高拥挤区间，提示可能存在拥挤或踩踏风险。

## 第八章 结论

### 8.1 核心发现

1. **A股板块轮动具有显著的跷跷板效应和四种主体风格轮换规律**（蓝筹/概念/周期/非周期），资金循环的"六道轮回"是深层驱动力
2. **经济周期是板块轮动的根本驱动力**，美林时钟和库存周期框架对行业配置具有重要指导意义
3. **六大量化轮动模型各有优劣**：动量模型趋势市中表现好，均值回归模型震荡市中表现好，宏观因子模型适合中长期配置，多策略融合模型最稳健
4. **均线能量+过滤+缓冲构成最实用的轮动框架**，年化24.9%、收益回撤比0.66的回测表现优异
5. **ETF轮动是散户参与板块轮动的最佳工具**，交易成本低、流动性好、覆盖面广
6. **拥挤度监控是风控关键**，高拥挤行业往往预示短期顶部

### 8.2 实操建议

- **入门**：从单ETF动量轮动开始，选择3-5只代表性ETF
- **进阶**：构建多因子综合得分模型，加入过滤和缓冲机制
- **实战**：结合经济周期判断，月度确定优势板块，周度执行调仓
- **持续**：建立行业拥挤度监控系统，定期回顾策略有效性
