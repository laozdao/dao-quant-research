## 一、引言：北交所的投资价值

### 1.1 北交所定位

北交所是服务创新型中小企业的主阵地：

| 定位 | 说明 | 投资意义 |
|------|------|---------|
| **专精特新** | 专注细分领域 | 隐形冠军 |
| **中小企业** | 市值小、成长空间大 | 高成长 |
| **转板预期** | 可向科创板/创业板转板 | 估值提升 |
| **流动性改善** | 投资者门槛降低 | 流动性提升 |

### 1.2 北交所企业特点

| 特点 | 说明 | 投资影响 |
|------|------|---------|
| **细分领域龙头** | 在细分市场有竞争力 | 护城河 |
| **成长性好** | 营收增速平均>20% | 成长投资 |
| **市值小** | 平均市值<50亿 | 弹性大 |
| **流动性差** | 日均成交额较低 | 进出困难 |
| **研究覆盖少** | 券商覆盖率低 | 信息不对称 |

## 二、北交所企业评估框架
