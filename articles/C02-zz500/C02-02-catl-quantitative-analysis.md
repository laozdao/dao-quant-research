# 宁德时代深度量化分析

## 摘要

宁德时代是全球动力电池龙头，也是新能源产业链的核心标的。本文运用DAO量化研究双引擎四层模型，对宁德时代进行全面量化分析，评估其投资价值。

## 一、公司概况

### 1.1 基本信息

| 项目 | 内容 |
|------|------|
| 公司全称 | 宁德时代新能源科技股份有限公司 |
| 股票代码 | 300750.SZ |
| 所属行业 | 电池制造 |
| 上市日期 | 2018年6月11日 |
| 总股本 | 44.0亿股 |

### 1.2 主营业务

- **动力电池**：三元锂电池、磷酸铁锂电池
- **储能电池**：发电侧、电网侧、用户侧储能
- **电池材料**：正极材料、前驱体

### 1.3 行业地位

- 全球动力电池市占率37%，连续7年第一

- 技术领先：CTP、麒麟电池、钠离子电池
