# 银行业量化分析框架：特色指标与因子适配

> **一句话摘要**：针对银行业特殊商业模式，构建包含净息差、不良率、资本充足率等特色指标的量化评估框架。

---

## 一、引言：银行业的特殊性

### 1.1 银行业商业模式特点

银行业是经营风险的行业，具有与其他行业截然不同的商业模式：

| 特征 | 说明 | 对分析的影响 |
|------|------|-------------|
| **高杠杆经营** | 资产负债率通常90%以上 | 传统负债率指标失效 |
| **强周期性** | 盈利与经济周期高度相关 | 需结合宏观周期分析 |
| **严监管** | 受央行、银保监会多重监管 | 监管政策影响重大 |
| **利差驱动** | 主要利润来自存贷利差 | 净息差是核心指标 |
| **风险滞后** | 资产质量风险暴露滞后 | 需关注前瞻指标 |

### 1.2 传统估值方法的局限

传统PE、PB估值方法在银行业应用时存在局限：

| 传统指标 | 局限 | 改进方向 |
|---------|------|---------|
| **PE估值** | 利润波动大，受拨备影响 | 结合PB和股息率 |
| **ROE分析** | 高杠杆扭曲ROE | 补充ROA分析 |
| **负债率** | 行业特性导致普遍偏高 | 改用资本充足率 |

## 二、银行业特色指标体系

### 2.1 盈利能力指标

#### 2.1.1 净息差（NIM）

净息差是银行核心盈利能力指标：

$$
\text{净息差} = \frac{\text{利息收入} - \text{利息支出}}{\text{生息资产平均余额}} \times 100\%
$$

| NIM水平 | 评价 | 代表银行 |
|--------|------|---------|
| > 2.5% | 优秀 | 招商银行、平安银行 |
| 2.0%-2.5% | 良好 | 股份制银行平均水平 |
| 1.8%-2.0% | 一般 | 国有大行平均水平 |
| < 1.8% | 较差 | 需关注 |

#### 2.1.2 净利差

$$
\text{净利差} = \text{生息率} - \text{付息率}
$$

净利差反映银行定价能力和负债成本控制能力。

#### 2.1.3 成本收入比

$$
\text{成本收入比} = \frac{\text{业务及管理费}}{\text{营业收入}} \times 100\%
$$

| 成本收入比 | 评价 |
|-----------|------|
| < 25% | 优秀 |
| 25%-30% | 良好 |
| 30%-35% | 一般 |
| > 35% | 较差 |

### 2.2 资产质量指标

#### 2.2.1 不良贷款率

$$
\text{不良贷款率} = \frac{\text{不良贷款余额}}{\text{贷款总额}} \times 100\%
$$

| 不良率 | 风险等级 | 评价 |
|--------|---------|------|
| < 1.0% | 低风险 | 优秀 |
| 1.0%-1.5% | 中低风险 | 良好 |
| 1.5%-2.0% | 中等风险 | 需关注 |
| > 2.0% | 高风险 | 警惕 |

#### 2.2.2 关注类贷款占比

关注类贷款是不良贷款的前瞻指标：

$$
\text{关注类贷款占比} = \frac{\text{关注类贷款}}{\text{贷款总额}} \times 100\%
$$

关注类+不良类贷款占比更能反映真实资产质量。

#### 2.2.3 拨备覆盖率

$$
\text{拨备覆盖率} = \frac{\text{贷款减值准备}}{\text{不良贷款余额}} \times 100\%
$$

| 拨备覆盖率 | 安全边际 | 监管要求 |
|-----------|---------|---------|
| > 300% | 很高 | — |
| 200%-300% | 充足 | 150% |
| 150%-200% | 达标 | 最低要求 |
| < 150% | 不足 | 不达标 |

#### 2.2.4 拨贷比

$$
\text{拨贷比} = \frac{\text{贷款减值准备}}{\text{贷款总额}} \times 100\%
$$

监管要求不低于2.5%。

### 2.3 资本充足指标

#### 2.3.1 资本充足率

$$
\text{资本充足率} = \frac{\text{总资本}}{\text{风险加权资产}} \times 100\%
$$

| 资本充足率 | 评价 | 监管要求 |
|-----------|------|---------|
| > 14% | 优秀 | — |
| 12%-14% | 良好 | 10.5% |
| 10.5%-12% | 达标 | 最低要求 |
| < 10.5% | 不足 | 不达标 |

#### 2.3.2 一级资本充足率

核心资本充足程度，监管要求不低于8.5%。

#### 2.3.3 核心一级资本充足率

最严格的资本充足指标，监管要求不低于7.5%。

### 2.4 成长能力指标

#### 2.4.1 规模增长指标

| 指标 | 计算公式 | 评价标准 |
|------|---------|---------|
| 总资产增速 | 总资产同比增长率 | 10%-15%为稳健 |
| 贷款增速 | 贷款总额同比增长率 | 与M2增速匹配 |
| 存款增速 | 存款总额同比增长率 | 与贷款增速匹配 |

#### 2.4.2 非息收入占比

$$
\text{非息收入占比} = \frac{\text{非利息收入}}{\text{营业收入}} \times 100\%
$$

非息收入占比反映银行业务多元化程度：

| 占比 | 评价 |
|------|------|
| > 40% | 优秀，中间业务能力强 |
| 30%-40% | 良好 |
| 20%-30% | 一般 |
| < 20% | 较差，过度依赖息差 |

## 三、银行业量化评分体系

### 3.1 评分维度与权重

| 维度 | 权重 | 核心指标 |
|------|------|---------|
| **盈利能力** | 30% | 净息差、ROA、成本收入比 |
| **资产质量** | 30% | 不良率、拨备覆盖率、关注类占比 |
| **资本充足** | 20% | 资本充足率、核心一级资本充足率 |
| **成长能力** | 15% | 营收增速、利润增速、非息收入占比 |
| **估值水平** | 5% | PB、股息率 |

### 3.2 单因子评分标准

#### 3.2.1 净息差评分

| NIM | 评分 |
|-----|------|
| > 2.5% | 90-100 |
| 2.2%-2.5% | 80-89 |
| 1.9%-2.2% | 70-79 |
| 1.6%-1.9% | 50-69 |
| < 1.6% | 0-49 |

#### 3.2.2 不良率评分

| 不良率 | 评分 |
|--------|------|
| < 1.0% | 90-100 |
| 1.0%-1.3% | 80-89 |
| 1.3%-1.6% | 70-79 |
| 1.6%-2.0% | 50-69 |
| > 2.0% | 0-49 |

#### 3.2.3 拨备覆盖率评分

| 拨备覆盖率 | 评分 |
|-----------|------|
| > 400% | 90-100 |
| 300%-400% | 80-89 |
| 200%-300% | 70-79 |
| 150%-200% | 50-69 |
| < 150% | 0-49 |

## 四、银行业分类分析

### 4.1 国有大行

| 特征 | 说明 |
|------|------|
| 代表 | 工商银行、建设银行、农业银行、中国银行、交通银行、邮储银行 |
| 优势 | 网点广、客户基础好、资金成本低 |
| 劣势 | 机制不够灵活、创新动力不足 |
| 特色指标 | 关注存款成本、普惠金融占比 |

### 4.2 股份制银行

| 特征 | 说明 |
|------|------|
| 代表 | 招商银行、平安银行、兴业银行、中信银行等 |
| 优势 | 机制灵活、创新能力强 |
| 劣势 | 负债成本较高、对公业务竞争激烈 |
| 特色指标 | 关注零售业务占比、财富管理收入 |

### 4.3 城商行

| 特征 | 说明 |
|------|------|
| 代表 | 北京银行、上海银行、江苏银行等 |
| 优势 | 区域优势明显、地方政府支持 |
| 劣势 | 区域集中风险、规模较小 |
| 特色指标 | 关注区域经济、地方债务风险 |

### 4.4 农商行

| 特征 | 说明 |
|------|------|
| 代表 | 渝农商行、沪农商行等 |
| 优势 | 下沉市场、三农业务 |
| 劣势 | 风控能力较弱、资产质量压力大 |
| 特色指标 | 关注涉农贷款占比、县域覆盖率 |

## 五、银行业投资建议

### 5.1 选股逻辑

| 优先级 | 选股标准 | 说明 |
|--------|---------|------|
| 1 | 资产质量优秀 | 不良率<1.3%，拨备覆盖率>300% |
| 2 | 盈利能力强 | NIM>2.0%，ROA>0.9% |
| 3 | 资本充足 | 核心一级资本充足率>9% |
| 4 | 估值合理 | PB<1.0倍，股息率>5% |

### 5.2 风险提示

| 风险类型 | 说明 | 监测指标 |
|---------|------|---------|
| **信用风险** | 经济下行导致不良上升 | 不良率、关注类占比 |
| **利率风险** | 利率下行压缩息差 | NIM、生息率 |
| **政策风险** | 监管政策变化 | 资本要求、信贷政策 |
| **流动性风险** | 存款流失 | 存贷比、流动性覆盖率 |

## 六、结语

银行业分析需要专门的框架和指标。通过净息差、不良率、拨备覆盖率、资本充足率等特色指标，可以更准确地评估银行的经营状况和投资价值。
