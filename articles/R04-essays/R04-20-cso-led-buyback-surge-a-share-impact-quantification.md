## 第1章 引言与核心摘要

### 1.1 事件概述

2026年7月20日，A股市场迎来一场罕见的央企领衔回购增持潮。当日超过30家上市公司集中发布回购或增持公告，涉及金额超过百亿元人民币。这一事件并非偶然的个股行为，而是一场由央企控股股东、国有资本运营公司系统协调推动的"国家队"入场行动，其规模和协调性在A股历史上亦属罕见。

从参与主体的层级来看，本次回购增持潮呈现出鲜明的"自上而下"特征。顶层是两大国有资本运营公司——中国国新和中国诚通，合计已投入近600亿元进行股票增持和ETF买入；中层是央企控股股东，包括三峡集团增持三峡能源15-30亿元、中铝集团增持中国铝业10-20亿元、中国建筑控股股东增持5-10亿元等；底层是龙头上市公司的董事长提议回购和自主回购，涵盖国电南瑞、三一重工、华友钴业等各行业龙头。

### 1.3 政策背景与信号意义

本次回购增持潮发生在央行3000亿元股票回购增持专项再贷款政策出台的背景下，是政策红利从"文件"到"落地"的关键一步。两大国有资本运营公司率先大规模使用再贷款资金，为全市场起到了示范效应。

从历史经验来看，A股历次大规模回购增持潮均出现在市场底部区域附近。本次央企系统性的集中回购增持，释放了强烈的"政策底→市场底"传导信号。我们将在后续章节中通过量化实证数据，系统验证回购增持事件与市场底部的关系。

## 第2章 回购增持的政策框架与制度演进

### 2.1 股票回购增持专项再贷款政策

2024年9月，中国人民银行创设股票回购增持专项再贷款工具，首期额度3000亿元，支持上市公司和主要股东回购增持股票。该政策的核心机制为：央行向商业银行发放再贷款，商业银行按照不超过2.25%的利率向符合条件的上市公司或其控股股东发放贷款，专项用于回购或增持股票。

**表2-1 股票回购增持专项再贷款政策要素**

| 要素 | 内容 |
|------|------|
| 政策名称 | 股票回购增持专项再贷款 |
| 创设时间 | 2024年9月 |
| 额度 | 3000亿元 |
| 贷款利率 | 不超过2.25% |
| 贷款期限 | 1-3年 |
| 适用对象 | 上市公司、控股股东 |
| 资金用途 | 专项用于回购或增持股票 |
| 首期申请规模 | 约2000亿元 |

截至2026年7月，中国国新旗下国新投资已使用该项再贷款及配套资金超过500亿元，成为再贷款政策最大的受益者和执行者之一。中国诚通也已累计买入近百亿元中国股票资产。两大国有资本运营公司的积极行动，为全市场树立了标杆。

### 2.2 回购制度优化历程

2024年以来，中国股票回购制度经历了多次重大优化，为本次回购增持潮奠定了制度基础。

**表2-2 2024年以来回购制度重大优化一览**

| 时间 | 政策/制度 | 核心内容 | 影响范围 |
|------|----------|---------|---------|
| 2024年3月 | 证监会"新国九条"配套规则 | 简化回购程序，取消窗口期限制 | 全市场 |
| 2024年5月 | 沪深交易所回购规则修订 | 放宽回购条件，缩短决策周期 | 沪深两市 |
| 2024年7月 | 新《公司法》正式实施 | 扩大回购事由，简化注销程序 | 全市场 |
| 2024年9月 | 央行再贷款政策创设 | 3000亿专项再贷款支持回购增持 | 全市场 |
| 2024年10月 | 国资委央企考核优化 | 市值管理纳入央企负责人考核 | 央企 |
| 2025年1月 | 证监会回购指引更新 | 鼓励"注销式回购"，强化披露要求 | 全市场 |
| 2025年6月 | 再贷款额度扩容 | 额度从3000亿扩大至5000亿 | 全市场 |

### 2.3 新《公司法》对回购的放宽

2024年7月1日起施行的新《公司法》对股份回购制度进行了重大修订，主要体现在以下方面：

**第一，扩大可回购情形。** 新法在原有六种回购情形基础上，进一步放宽了"为维护公司价值及股东权益所必需"条款的适用范围，使得上市公司在股价低于每股净资产或低于行业平均市盈率时，可以更加灵活地进行回购。

**第二，简化回购决策程序。** 新法明确公司章程或股东大会授权董事会决定回购方案的，董事会可以依法快速决策，缩短了从"提议"到"实施"的时间周期。这解释了为何本次潮中多家公司采用"董事长提议回购"的方式。

**第三，放宽回购股份处置限制。** 新法对回购股份的转让、注销等处置方式给予了更大灵活性，特别是在员工持股计划和股权激励方面，减少了制度性约束。

**表2-3 新旧《公司法》回购制度对比**

| 比较维度 | 旧法 | 新法 |
|---------|------|------|
| 可回购情形 | 6种 | 6种（放宽适用解释） |
| 董事会权限 | 需股东大会授权 | 章程或股东大会可授权 |
| 回购决策周期 | 通常需1-2个月 | 可缩短至2-4周 |
| 注销程序 | 较为繁琐 | 简化注销流程 |
| 回购股份上限 | 不超已发行股份10% | 维持10%上限 |
| 回购资金来源 | 自有资金 | 自有资金+专项贷款 |
| 员工持股计划转让 | 限制较多 | 放宽转让条件 |

### 2.4 央企考核体系变化

2024年10月，国资委正式将"市值管理"纳入央企负责人经营业绩考核体系。这一制度变革对央企回购增持行为产生了深远影响。

**考核权重调整：** 市值管理指标在央企负责人考核中的权重约为10-15%，与净利润增长率、ROE等传统指标并列。这意味着央企负责人有了内在动力去维护和提升市值。

**考核方式：** 市值管理考核并非简单看股价涨跌，而是综合评估市盈率、市净率、股息率等指标的改善幅度，以及分红、回购、增持等市值管理工具的使用情况。

**表2-4 央企负责人经营业绩考核指标权重变化**

| 考核维度 | 2023年权重 | 2026年权重 | 变化幅度 |
|---------|-----------|-----------|---------|
| 净利润增长率 | 20% | 18% | -2% |
| ROE（净资产收益率） | 15% | 13% | -2% |
| 营业收入增长率 | 10% | 10% | 0% |
| **市值管理** | **0%** | **12%** | **+12%** |
| 创新发展 | 15% | 15% | 0% |
| 风险控制 | 10% | 10% | 0% |
| ESG与社会责任 | 10% | 12% | +2% |
| 安全生产 | 10% | 5% | -5% |
| 其他 | 10% | 5% | -5% |

### 2.5 回购vs增持的监管差异与政策意图

回购和增持虽然都能传递看好信号，但二者在监管框架和政策意图上存在重要差异。

**表2-5 回购与增持的监管差异对比**

| 比较维度 | 股份回购 | 股东增持 |
|---------|---------|---------|
| 执行主体 | 上市公司自身 | 控股股东/董监高 |
| 资金来源 | 公司自有资金或专项贷款 | 股东自有资金或贷款 |
| 审批程序 | 董事会/股东大会 | 无需上市公司审批 |
| 股份处理 | 注销/员工持股/激励 | 直接持有 |
| 对EPS影响 | 注销式回购直接增厚 | 无直接影响 |
| 信息披露 | 需公告回购方案和进展 | 需公告增持计划 |
| 窗口期限制 | 定期报告前有窗口期 | 同样有窗口期 |
| 信号强度 | 中等（公司层面） | 较强（股东用真金白银） |
| 政策优先支持 | 较高（再贷款直接对接） | 同样受再贷款支持 |
