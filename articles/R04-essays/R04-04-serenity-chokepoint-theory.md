# Serenity 瓶颈投资方法论深度研究：紫苏叶理论与供应链逆向拆解

> **一句话摘要**：不买英伟达，只买英伟达离不开的公司——从底层材料出发逆向推演，在供应链最不起眼的瓶颈处寻找认知差。

---

## 一、引言/背景

### 1.1 问题提出

2026年5月，一位自称拒绝过英伟达AI团队负责人offer的匿名投资者Serenity，在X平台晒出2026年迄今4502.45%的账面回报。据BlockBeats报道，这位前Reddit WSB板块的资深交易员，粉丝已逼近40万，其研究被Bloomberg和路透社引用。他从不买英伟达，而是专挖AI产业链里"缺了它就停摆"的隐形节点。

Serenity的身份至今是个谜。二次元白发女性头像，从不露脸，从不接受采访。他自称前AI研究科学家、Nature论文作者、RISC-V基金会成员。按他自己的说法，2018年英伟达股价才6美元时，他拒绝过对方AI团队负责人的offer。没人能核实这些履历，但战绩摆在那里，质疑声越来越小。

几个数字挺有意思：
- 2025年下半年，6个月回报率630%
- 2026年至今（截至5月28日），回报率4502.45%，也就是45倍
- 过去两年累计22562%，225倍
- 公开推荐的35只股票，只有4只下跌

但比数字更值得研究的，是他的方法论。

### 1.2 研究目标

本文旨在：
1. 系统梳理Serenity的核心投资方法论（Chokepoint Theory / 紫苏叶理论）
2. 拆解其AI产业链七层分析框架
3. 深度分析代表性案例的投资逻辑
4. 提炼可复制的研究框架与检查清单

> **重要提示**：Serenity的身份背景、收益数据均为自我声明，未经独立验证。本文研究目的是提炼可复制的投资思维框架，而非推荐具体标的。

## 二、核心方法论：Chokepoint Theory

### 2.1 方法论本质

Serenity策略并非赌"AI会发展"，而是赌产业链中**最难替代、扩产最慢的物理环节**。

> **核心逻辑链条**：  
> 需求爆发 → 逆向拆解BOM → 锁定不可绕过的衬底/外延/特种设备 → 聚焦小市值、高纯度公司

赚的是市场对"隐形瓶颈"的**认知差**和**信息转化效率差**。

### 2.2 与主流投资方法的区别

| 维度 | 主流方法 | Serenity 方法 |
|------|----------|---------------|
| 分析方向 | 自上而下（终端产品 → 上游） | 自下而上（原材料 → 终端） |
| 选股标准 | 公司好不好？ROE高不高？ | 产业链哪里最容易卡住？ |
| 关注对象 | 明星公司（英伟达、微软） | 隐形供应商（衬底、激光器） |
| 竞争优势 | 护城河、管理层 | 不可替代性、产能稀缺性 |
| 信息来源 | 财报、研报 | 专利、供应链协议、GitHub、论坛 |

### 2.3 Chokepoint 的三个判定标准

1. **全球仅有极少数供应商**能提供该环节的产品或服务
2. **下游巨头对该环节有刚性依赖**，一旦断供整条产业链停摆
3. **扩产周期极长**（通常按年计算），短期内格局无法改变

### 2.4 核心投资哲学

> "我不是在预测哪个股票会涨，我是在追踪钱会流到哪里去。"  
> —— Serenity

> "不要只看谁在卖铲子，要找谁卡住了铲子的关键零件。"  
> —— Serenity

## 三、紫苏叶理论：比喻与映射

### 3.1 比喻来源

Serenity将自己的投资方法称为"紫苏叶理论"，比喻来自东京顶级寿司店：

- **食客最追捧的**：金枪鱼大腹（人人皆知的热门标的）
- **整盘寿司能不能上桌的关键**：那片来自伊豆半岛特定小农场的紫苏叶（不起眼的瓶颈环节）

> "没了金枪鱼，菜单少几道菜；没了紫苏叶，整家店就得关门。"

### 3.2 产业链映射

| 寿司店 | AI 产业链 |
|--------|-----------|
| 金枪鱼大腹 | 英伟达、微软、谷歌、Meta |
| 紫苏叶 | InP衬底、CPO激光器、高纯度磷原料、特种气体 |
| 寿司店 | AI数据中心、光通信网络 |

### 3.3 投资启示

Serenity **从不买英伟达**，专门找那些"一旦断供，整个AI产业就得停摆"的隐形供应商。这种思维方式的本质是：**在所有人追逐显性赢家时，去寻找支撑整个系统的隐性瓶颈**。

## 四、投资逻辑链条：逆向拆解供应链

### 4.1 四步研究流程

```
Step 1: 需求爆发确认
    ↓ 验证下游资本开支趋势
Step 2: 逆向拆解 BOM（物料清单）
    ↓ 从终端产品逐层向上追溯
Step 3: 锁定物理瓶颈
    ↓ 识别不可替代、扩产最慢的环节
Step 4: 聚焦小市值高纯度公司
    ↓ 在机构资金到达之前建仓
```

### 4.2 关键研究动作

1. **从超大规模数据中心资本开支出发**，倒推数据中心扩张节奏
2. **推到带宽和功耗的物理边界**，识别技术路线的硬约束
3. **定位最窄的瓶颈环节**，分析全球产能分布
4. **啃材料学论文**，分析专利布局
5. **梳理产能扩张计划**，追踪供应商认证周期
6. **关注出口管制动态**，识别地缘政治风险溢价

### 4.3 信息来源矩阵

| 信息层级 | 具体来源 | 用途 |
|----------|----------|------|
| 学术层 | 材料学论文、专利数据库 | 技术路线验证 |
| 社区层 | GitHub、开发者论坛、采购讨论 | 需求预判（如RPI案例） |
| 供应链层 | 供应商公告、合作伙伴页面、招聘信息 | 客户映射验证 |
| 产业层 | Conference、产品路线图 | 叙事吸收时间差 |
| 政策层 | 出口管制、贸易政策 | 地缘政治风险溢价 |

## 五、AI 产业链七层拆解框架

Serenity将AI光通信和硅光子供应链拆分为七个关键层级，每一层都对应着具体的瓶颈点和投资标的。

### 5.1 七层框架总览

```
Layer 7: 光纤互联（最终应用）
    ↑
Layer 6: 光收发器组装、测试设备
    ↑
Layer 5: 连续波激光器（CPO光源）
    ↑
Layer 4: 磷化铟衬底制造
    ↑
Layer 3: 热解氮化硼坩埚（单一供应商设备瓶颈）
    ↑
Layer 2: 稀有金属原料采炼
    ↑
Layer 1: 高纯特种气体、金属有机源
```

### 5.2 各层瓶颈分析

#### Layer 1-2: 原材料层

- **关键物料**：高纯铟、磷原料、金属有机源（MO源）
- **瓶颈特征**：关键物料集中在一两座工厂，出口管制或运输受限即可卡住区域流片
- **地缘政治**：中国掌控全球50%铟储量与70%产量，2025年2月起实施出口配额管理
- **价格影响**：精铟价格从2025年末2800元/千克飙升至2026年3月4950元/千克

#### Layer 3: 设备层

- **关键设备**：热解氮化硼（PBN）坩埚、MBE（分子束外延）设备
- **瓶颈特征**：全球仅少数供应商，交期超长
- **投资逻辑**：设备产能直接决定上游材料扩产速度

#### Layer 4: 衬底层

- **关键材料**：磷化铟（InP）衬底、SOI衬底
- **瓶颈特征**：
  - 全球90%以上产能被日本住友、美国AXT、日本JX金属三家垄断
  - 从建厂到量产周期按年计算
  - 2026年全球需求预计260-300万片，有效产能仅75万片左右，缺口70%+
- **代表标的**：AXTI（AXT Inc.）

#### Layer 5: 光源层

- **关键器件**：连续波（CW）激光器、大功率激光器芯片
- **瓶颈特征**：
  - CPO需要大功率CW激光器，核心是InP衬底及外延片
  - 全球高度集中、扩产慢
  - 近期外延片交期持续拉长，直接映射为光模块厂产能焦虑
- **代表标的**：SIVE（Sivers Semiconductors）、LITE（Lumentum）、COHR（Coherent）

#### Layer 6: 组装测试层

- **关键环节**：光收发器组装、测试设备
- **瓶颈特征**：先进封装设备（如临时键合/解键合设备）全球仅两三家供应商，交期超一年
- **投资逻辑**：CoWoS扩产受限于TSV刻蚀设备、临时键合/解键合设备

#### Layer 7: 系统层

- **关键应用**：数据中心光互联、CPO（共封装光学）
- **瓶颈特征**：
  - 单柜功率向200kW演进
  - 液冷分配单元（CDU）、冷板微通道加工产能远跟不上GPU节奏
  - 扩产周期12-18个月，成为大规模部署的物理锁死点

### 5.3 其他关键赛道

#### NeoCloud（二级云服务商）

- **核心思路**：在超大规模云计算公司大举投入AI基础设施资本支出的背景下，寻找承接这些资本支出的二级受益公司
- **特别关注**：从比特币矿企转型为AI算力服务商的一批公司
- **代表标的**：NBIS（Nebius）、CRWV、WULF、CIFR、IREN
- **Serenity定义**：Nebius是"NeoCloud赛道的首选"，"最纯粹的AI基础设施不对称机会"

#### AI 电力链

- **核心逻辑**：电力与散热从成本项变为扩容阈值
- **关键环节**：电网、燃机、核电、功率半导体
- **代表标的**：XLU、VST、CEG

## 七、AI 辅助研究流程

### 7.1 Serenity 的 AI 使用方式

> "我不是用AI炒股，而是用AI制造'研究差'。"

### 7.2 具体流程

1. **输入所有信息源**：行业报告、学术论文、专利、财报
2. **多模型辩论**：让多个AI模型互相辩论找漏洞
3. **构建供应链地图**：系统化梳理产业链层级关系
4. **对抗性压力测试**：模拟"投资委员会答辩"效果，在发布前找出逻辑漏洞

### 7.3 与散户使用 AI 的根本区别

| 维度 | 散户用法 | Serenity 用法 |
|------|----------|---------------|
| 目的 | 寻找确认（"这只股票能不能买"） | 寻找反驳（"我的逻辑哪里错了"） |
| 结果 | 强化确认偏误 | 消除确认偏误 |
| 价值 | 信息获取 | 研究差制造 |

> **关键洞察**：普通散户用AI找"这只股票能不能买"，Serenity用AI找"我的逻辑哪里错了"。这个区别，可能是散户和专业研究者之间最大的鸿沟。

## 八、风险分析与方法边界

### 8.1 核心风险

#### 1. 技术替代风险（最大隐形风险）

- InP激光器可能被薄膜铌酸锂或量子点激光器部分取代
- 物理瓶颈会在一夜之间松动
- 技术路线替代或政策突变可能导致腰斩回撤

#### 2. 流动性风险

- 小市值个股日成交仅数百万美元
- 理想换仓与实际滑点差距巨大
- 暴跌时流动性枯竭

#### 3. 幸存者偏差

- 公开讨论只展示成功案例
- 技术路线替代或政策突变导致的腰斩回撤很少被同等传播

#### 4. 数据未审计

- 收益为"市场流传口径"，存在选择性披露或回测过拟合
- 实盘复现概率极低

#### 5. 信息差窗口急剧缩短

- 社交网络和内卷调研使"卡脖子"小盘股被迅速定价
- 先发优势不可复制

#### 6. "Serenity 效应"本身成为风险

- 散户往往在消息传开、股价暴涨后才跟风买入
- 做空机构已经盯上：Sivers约17%可流通股已被借出做空（3月初才1.6%）

### 8.2 方法边界

| 边界 | 说明 |
|------|------|
| **能力圈边界** | 超额收益几乎完全来自AI光通信和半导体供应链领域。在加密货币、金融科技、社交平台等方向上，推荐甚至劣于被动指数 |
| **规模边界** | 持仓高度集中于流动性差的中小盘科技股，牛市封神，熊市可能毁灭 |
| **可复制性边界** | 粉丝规模本身正在改变游戏规则。40万粉丝和3.7万付费订阅者使帖子本身成为市场事件 |
| **时间边界** | 策略可复制性正在下降。机构算法已大量跟踪Serenity的公开推文，会提前抢跑定价 |

### 8.3 回测数据揭示的真相

- **fxcryptobots回测**：79%的推荐跑输基准，仅21%的标的产生超额收益
- **真正创造巨大回报的**：集中在光子学和CPO领域的约20个标的（AXTI、AAOI、NBIS、MU、AEHR、LITE等），实现2-7倍回报
- **表现惨淡的方向**：加密货币和金融科技（HIMS -47%、MSTR -54%、COIN -43%）
- **关键结论**：Serenity的"边际优势"（alpha）高度集中在AI光通信和半导体供应链这一个垂直领域

### 8.4 核心争议

#### "高智商包装的拉高出货"

- 批评逻辑：在微型股上建仓 → 向几十万粉丝发布详尽投资论文 → 散户跟风买入推高股价 → 何时卖出、卖出多少外界无从知晓
- Serenity的回应：曾在某条帖子中坦承"最近表现不太好了"（Not doing so well anymore）

#### 匿名性的双刃剑

- **优势**：不受机构合规限制，可以自由覆盖全球任何市值的公司，填补研究真空
- **风险**：没有声誉约束，没有监管审查，没有利益冲突披露的义务；买入/卖出时间点是黑箱

## 十、可复制的研究框架

### 10.1 四问筛选法

看到一个AI主题，先问四个问题：

1. **这条产业链里真正短缺的东西是什么？**
2. **下游愿意为这个瓶颈付多高价格？**
3. **哪个小公司最接近这个瓶颈，而且市场还没完全定价？**
4. **这个公司有没有融资、稀释、客户集中、量产失败的硬伤？**

如果这四个问题都能过，再进入财报、估值、流动性和仓位层面的研究。

### 10.2 研究检查清单

#### 产业链拆解

- [ ] 逆向拆解BOM，识别每一层的关键物料/设备/工艺
- [ ] 绘制全球产能分布地图
- [ ] 识别全球仅有1-3家供应商的关键环节
- [ ] 分析扩产周期（是否按年计算？）
- [ ] 评估技术替代风险（是否有替代路线？替代概率？）

#### 公司研究

- [ ] 市值是否小于10亿美元（机构研究真空区）？
- [ ] 客户质量如何？是否有头部客户背书？
- [ ] 合约/订单可见性如何？
- [ ] 融资结构是否健康？是否存在ATM稀释风险？
- [ ] GAAP margin趋势如何？
- [ ] 量产失败风险？

#### 时机判断

- [ ] 叙事吸收时间差是否存在？（Conference/合作伙伴页面/供应商公告/招聘信息 → Sell-side/机构仓位反应）
- [ ] 是否处于qualification/design win/capacity tightness/客户映射刚出现的窗口？
- [ ] 估值是否已透支未来预期？

#### 风险管理

- [ ] 单票仓位上限（建议不超过5-10%）
- [ ] 流动性评估（日成交量是否足够？）
- [ ] 技术路线突变应急预案
- [ ] 止损/止盈机制

## 十一、结论与启示

### 11.1 值得学习的三件事

1. **逆向拆解的思维习惯**  
   大多数人看AI投资，盯着GPU、大模型、算力。Serenity的思路是：既然所有人都盯着龙头，那龙头的供应商里，谁是不可替代的？谁一旦断供，整条链都会瘫痪？这种"从终端需求倒推瓶颈"的思维方式，不仅适用于AI，也适用于新能源、半导体、医药等任何有清晰产业链的行业。

2. **用AI验证而非用AI预测**  
   Serenity不是让AI帮他选股，而是让AI扮演对手方，攻击自己的逻辑。这比99%的人用AI的方式都高明——大多数人在用AI寻找确认偏误，他在用AI消除确认偏误。

3. **在无人关注处寻找价值**  
   他选的标的普遍市值小、无机构覆盖、散户看不懂。这正是超额收益的来源：当华尔街分析师还没来得及研究的时候，价格里还没有反映价值。但这也意味着流动性差、波动大，不适合大资金，更不适合追高。

### 11.2 不能学的三件事

1. **单票重仓**  
   Serenity的超高收益很大程度来自极端仓位集中。这对普通投资者是致命的——一次判断失误就可能归零。

2. **忽视估值**  
   他不看PE，只看产能缺口。这种做法在产能紧缺的早期阶段有效，但当市场充分定价后，估值泡沫可能比产能缺口更危险。以SIVE为例，当前市销率60-80倍，已隐含至少20倍增长预期。若5个独立假设每个兑现概率80%，全部兑现的概率仅33%。

3. **盲目跟单**  
   机构算法已经大量跟踪Serenity的公开推文，会提前抢跑定价。普通人看到帖子时，价格可能已经被推高了。Serenity自己也说过：他的策略可复制性正在下降。

### 11.3 最终判断

Serenity的故事之所以迷人，是因为它同时满足了两种叙事：

- **她自己的叙事**：用极客的深度降维打击金融的广度，在供应链的缝隙里找到物理法则级别的确定性。这种叙事是理性主义的胜利：研究足够深，就能看到别人看不到的东西。
- **旁观者的叙事**：在AI泡沫的狂热中，又一个被神话的名字。225倍收益的截图、匿名的身份、无法验证的履历，这些都足以让理性的人保持警惕。

两种叙事都有道理。

但如果我们剥离所有神话色彩，只看方法论本身，Serenity的投资框架至少提供了一个有价值的启发：

> **在任何一个超级产业趋势里，最赚钱的位置往往不在聚光灯下，而在供应链最不起眼的瓶颈处。**

金枪鱼很贵，但没有紫苏叶，寿司店会关门。

这不只适用于AI。适用于每一个你正在关注的热门赛道。
