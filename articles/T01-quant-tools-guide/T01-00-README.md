## 学习路径推荐

### 入门路径（0-3个月）

```
Python基础 → NumPy → pandas → yfinance → Matplotlib
```

适合零基础的量化初学者，建议从Python语言基础开始，逐步掌握数据处理和可视化能力。

### 进阶路径（3-6个月）

```
pandas高级 → Backtrader回测 → scikit-learn → 因子研究
```

在掌握基础工具后，学习策略回测和简单的机器学习方法。

### 高级路径（6-12个月）

```
vectorbt → QuantConnect LEAN → 深度学习 → 生产部署
```

面向有经验的量化从业者，学习高性能回测、云端平台和生产级部署方案。

## 工具选型指南

### 按需求场景选择

| 场景 | 推荐工具组合 | 理由 |
|------|-------------|------|
| 个人研究学习 | Python + pandas + yfinance + Backtrader | 免费开源，社区活跃，学习资源丰富 |
| 因子研究与多参数优化 | Python + pandas + vectorbt | 向量化运算速度快，支持大规模参数扫描 |
| 多资产专业交易 | QuantConnect LEAN + Python | 云端执行，支持股票/期货/加密货币 |
| 统计套利策略 | R语言 + quantmod + rugarch | 统计模型丰富，计量经济学支持完善 |
| 机器学习量化策略 | Python + scikit-learn + pandas | 算法全面，与数据处理无缝集成 |

### 按技术栈选择

| 技术栈 | 适用人群 | 核心优势 |
|--------|---------|---------|
| 纯Python栈 | 大多数量化从业者 | 生态最完整，学习成本最低 |
| Python + R混合 | 统计学背景研究者 | 兼顾工程效率与统计深度 |
| C# + LEAN | 机构级开发者 | 高性能，与云端平台深度集成 |
