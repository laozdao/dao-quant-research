## 一、统一算子定义（先看懂符号）

- `CLOSE`/`OPEN`/`HIGH`/`LOW`/`VOLUME`：当日收盘价 / 开盘价 / 最高价 / 最低价 / 成交量

- `VWAP`：日内均价（成交额 / 成交量）

- `DELTA(A,n)`：A\(t\) − A\(t−n\)

- `DELAY(A,n)`：A\(t−n\)

- `SUM(A,n)`：过去 n 日求和

- `STD(A,n)`：过去 n 日标准差

- `CORR(A,B,n)`：过去 n 日 A 与 B 相关系数

- `RANK(A)`：当日截面升序排名（0\~1）

- `TSRANK(A,n)`：当前值在过去 n 日的时间序列排名

- `SMA(A,n,m)`：n 日简单移动平均，m 为权重

- `MAX(A,n)`/`MIN(A,n)`：过去 n 日最大 / 最小

- `COUNT(cond,n)`：过去 n 日满足条件 cond 的天数

- `REGBETA(A,B,n)`：A 对 B 的 n 日回归系数

- `REGRESI(A,B,n)`：A 对 B 回归的残差
